# 中医六经辨证检查清单

> 本清单用于临床逐项核对，避免漏项。建议打印随身携带。

---

## 使用说明

1. 每项打勾 ✓ 或打叉 ✗
2. 任意步骤发现问题，立即回头修正
3. 全部打勾后，诊断可靠

---

## 第一步：主诉与病史

| 检查项 | 状态 | 备注 |
|--------|------|------|
| 主诉已记录 | ☐ | 用患者原话 |
| 现病史已记录 | ☐ | 发病时间、诱因、演变 |
| 治疗史已询问 | ☐ | 曾用何药、效果如何 |
| 既往史已询问 | ☐ | 慢性病、过敏史、手术史 |

### 急危重症排查 ⚠️

| 检查项 | 状态 | 如有，立即就医 |
|--------|------|---------------|
| 无持续剧烈腹痛拒按 | ☐ | 急腹症 |
| 无高热神志改变 | ☐ | 重症感染 |
| 无突发胸痛大汗气短 | ☐ | 心梗 |
| 无突发剧烈头痛呕吐 | ☐ | 脑卒中 |
| 无呼吸困难喘促 | ☐ | 肺部急症 |
| 无呕血黑便 | ☐ | 上消化道出血 |
| 无但欲寐+肢冷+脉微欲绝 | ☐ | 少阴亡阳 |
| 无意识模糊抽搐 | ☐ | 颅内病变 |

**⚠️ 以上任意项有问题 → 先排除危险情况，再继续辨证**
