# 中医六经辨证流程图

> 本流程图展示辨证全流程及关键决策点。

---

## 一、完整辨证流程图

```mermaid
graph TD
    A[开始：患者就诊] --> B[第一步：主诉与病史]
    B --> C{急危重症排查}
    C -->|有危险信号| D[⚠️ 立即就医/排除危险]
    C -->|无危险| E[第二步：十问为纲]

    E --> F[第三步：舌诊]
    F --> G[第四步：脉诊]

    G --> H[第五步：定经]
    H --> H1[定表里]
    H1 --> H2[定寒热]
    H2 --> H3[定虚实]
    H3 --> H4[确定六经]

    H4 --> I[第六步：方证对勘]
    I --> I1{方证匹配?}
    I1 -->|不匹配| J[重新辨证/考虑合病]
    J --> I
    I1 -->|匹配| K[第七步：药证校验]

    K --> K1{药证有据?}
    K1 -->|无据| L[调整方药]
    L --> K
    K1 -->|有据| M[第八步：证据核查]

    M --> M1{证据≥5条?}
    M1 -->|<3条| N[重新辨证]
    N --> H
    M1 -->|3-4条| O[继续观察/注意合病]
    O --> P[第九步：反向验证]
    M1 -->|≥5条| P

    P --> P1{发现矛盾?}
    P1 -->|有矛盾| Q[合病排查/修正诊断]
    Q --> H
    P1 -->|无矛盾| R[第十步：鉴别诊断]

    R --> S[第十一步：合病排查]
    S --> S1{有合病?}
    S1 -->|有| T[调整选方思路]
    T --> U[第十二步：选方定药]
    S1 -->|无| U

    U --> V{禁忌排查}
    V -->|有禁忌| W[调整方药]
    W --> V
    V -->|无禁忌| X[确定处方]

    X --> Y[医嘱：煎服法+饮食禁忌]
    Y --> Z[复诊反馈]
    Z --> Z1{有效?}
    Z1 -->|有效| Z2[效不更方]
    Z1 -->|无效/加重| Z3[重新辨证]
    Z3 --> H

    style D fill:#ff6b6b
    style H4 fill:#4ecdc4
    style I1 fill:#ffe66d
    style K1 fill:#ffe66d
    style M1 fill:#ffe66d
    style P1 fill:#ffe66d
    style V fill:#ffe66d
```

## 二、定经决策图

```mermaid
graph TD
    A[问诊+舌脉信息] --> B{定表里}

    B -->|恶寒发热头痛脉浮| C[表证 → 太阳]
    B -->|吃喝拉撒睡问题为主| D[里证]
    B -->|往来寒热口苦胸胁苦满| E[半表半里 → 少阳]

    C --> F{定寒热}
    D --> F
    E --> F

    F -->|口渴喜凉尿黄舌红脉数| G[热证]
    F -->|口不渴喜热尿清舌淡脉迟| H[寒证]
    F -->|上热下寒/口苦便稀| I[寒热错杂 → 厥阴]

    G --> J{定虚实}
    H --> J
    I --> J

    J -->|声高气粗痛拒按脉有力| K[实证]
    J -->|声低气怯痛喜按脉无力| L[虚证]

    C --> M[六经确定]
    D --> M
    E --> M
    G --> M
    H --> M
    I --> M
    K --> M
    L --> M

    M --> N{结果}

    N -->|表+寒+实| O[太阳表实 → 麻黄汤]
    N -->|表+寒+虚| P[太阳表虚 → 桂枝汤]
    N -->|里+热+实+腑实| Q[阳明腑实 → 承气汤]
    N -->|里+热+实+无腑实| R[阳明经热 → 白虎汤]
    N -->|半表半里| S[少阳 → 小柴胡汤]
    N -->|里+寒+虚+脾为主| T[太阴 → 理中汤]
    N -->|里+寒+虚+肾为主+危候| U[少阴 → 四逆汤]
    N -->|寒热错杂| V[厥阴 → 乌梅丸]

    style O fill:#a8e6cf
    style P fill:#a8e6cf
    style Q fill:#ffd3b6
    style R fill:#ffd3b6
    style S fill:#dcedc1
    style T fill:#c9f4d4
    style U fill:#ff6b6b
    style V fill:#ffeaa7
```

```mermaid
    E -->|浮紧| F[麻黄汤证]
    E -->|非浮紧| G[重新辨证]
```

```mermaid
graph TD

    H -->|痞满为主| J[小承气汤]
```

## 五、太阴vs少阴鉴别流程

```mermaid
graph TD
    A[里寒虚证] --> B{精神状态}

    B -->|神志清楚| C[太阴]
    B -->|但欲寐/精神萎靡| D{脉象}

    C --> E[脾为主：腹满吐利]
    E --> F[理中汤]

    D -->|沉弱| G[太阴可能性大]
    D -->|微细欲绝| H[少阴]

    G --> I[进一步观察]
    H --> J{有无危候}

    J -->|肢冷+冷汗+脉微欲绝| K[少阴亡阳 ⚠️]
    J -->|无危候| L[少阴阳虚]

    K --> M[四逆汤急救+送医]
    L --> N[四逆汤/真武汤]

    style K fill:#ff6b6b
    style M fill:#ff6b6b
```

## 六、禁忌排查流程

```mermaid
graph TD
    A[选方后] --> B{方剂类型}

    B -->|含麻黄| C{患者情况}
    C -->|高血压/心脏病/失眠| D[⚠️ 禁用麻黄汤]
    C -->|体虚/老人/孕妇| E[⚠️ 慎用麻黄汤]
    C -->|无禁忌| F[可用]

    B -->|含大黄芒硝| G{患者情况}
    G -->|孕妇| H[⚠️ 禁用承气汤]
    G -->|体虚/老人/产后| I[⚠️ 慎用承气汤]
    G -->|无禁忌| J[可用]

    B -->|含附子| K{患者情况}
    K -->|真热假寒| L[⚠️ 禁用四逆汤]
    K -->|阴虚发热| M[⚠️ 禁用四逆汤]
    K -->|少阴寒证| N{煎法}
    N -->|附子先煎≥1小时| O[可用]
    N -->|未久煎| P[⚠️ 附子有毒需久煎]

    style D fill:#ff6b6b
    style E fill:#ffeaa7
    style H fill:#ff6b6b
    style I fill:#ffeaa7
    style L fill:#ff6b6b
    style M fill:#ff6b6b
    style P fill:#ff6b6b
```

## 七、复诊反馈流程

```mermaid
graph TD
    A[复诊] --> B{效果评估}

    B -->|有效| C{症状是否消失?}
    C -->|是| D[停药观察]
    C -->|否| E[效不更方，继服]

    B -->|无效| F{是否辨证正确?}
    F -->|需重新辨证| G[回到定经步骤]
    F -->|辨证正确| H{是否剂量不足?}
    H -->|是| I[调整剂量]
    H -->|否| J{是否煎服法不当?}
    J -->|是| K[调整煎服法]
    J -->|否| L[考虑合病/体质因素]

    B -->|加重| M{是否有禁忌?}
    M -->|有| N[调整方药]
    M -->|无| O[重新辨证]

    G --> P[重新走辨证流程]
    I --> Q[继续观察]
    K --> Q
    L --> P
    N --> Q
    O --> P

    style G fill:#ffeaa7
    style O fill:#ff6b6b
    style P fill:#dcedc1
```

---

## 使用说明

1. 本流程图使用 Mermaid 语法，可在支持 Mermaid 的 Markdown 编辑器中渲染
2. 绿色节点表示正常流程
3. 黄色节点表示决策点
4. 红色节点表示需要特别注意的危险信号或禁忌
