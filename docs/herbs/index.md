## 二、核心内容概览

### 三档分类体系

```
第一档（必进15味）：覆盖近90%用药频次
甘草 桂枝 生姜 大枣 芍药 半夏 干姜 人参 附子 茯苓 大黄 麻黄 白术 黄芩 柴胡

第二档（补充28味）：让经方体系完整
黄连 细辛 枳实 厚朴 石膏 当归 杏仁 五味子 葛根 龙骨 牡蛎 栀子 泽泻 猪苓
阿胶 麦冬 桃仁 牡丹皮 川芎 吴茱萸 黄芪 山药 山茱萸 熟地黄 天花粉 苏梗 连翘 桔梗

第三档（按需10味）：低频但不可替代
饴糖 粳米 芒硝 葶苈子 旋覆花 代赭石 赤石脂 升麻 防己 薏苡仁
```

---

### 六经主药速查

```
太阳主药：桂枝、麻黄
阳明主药：大黄、石膏
少阳主药：柴胡、黄芩、半夏
太阴主药：白术、干姜、人参、茯苓
少阴主药：附子、干姜
厥阴主药：当归、吴茱萸（第二档）

六经通用：甘草、大枣、生姜、芍药
```

---

### 一五九定律

**15%的药味，覆盖近90%的用药频次**

| 药味 | 出现次数 | 归经 | 药性 |
|------|----------|------|------|
| 甘草 | 123 | 心、肺、脾、胃 | 平 |
| 桂枝 | 70 | 心、肺、膀胱 | 温 |
| 生姜 | 67 | 脾、胃、肺 | 温 |
| 大枣 | 59 | 脾、胃 | 温 |
| 芍药 | 51 | 肝、脾 | 微寒 |
| 半夏 | 38 | 脾、胃、肺 | 温 |
| 干姜 | 36 | 脾、胃、心、肺 | 热 |
| 人参 | 34 | 脾、肺、心 | 微温 |
| 附子 | 34 | 心、肾、脾 | 热 |
| 茯苓 | 32 | 心、脾、肾 | 平 |

前10味药 → 占经方用药频次半数以上  
前15味药 → 覆盖经方近90%用药频次

### 预算参考

| 档次 | 预估费用 |
|------|----------|
| 第一档（15味） | ~1240元 |
| 第二档（28味） | ~1500元 |
| 第三档（10味） | ~300-500元 |
| **总计** | **3000-3700元** |

**对比**：普通中医诊所200味起步，首次进货动辄上万。经方路线不到三分之一。

### 存储管理要点

| 存放类别 | 药味 | 存放要求 |
|----------|------|----------|
| 毒性专柜 | 附子、生半夏 | 专柜、双人双锁、登记台账 |
| 矿物类单独 | 石膏、赤石脂、代赭石、龙骨、牡蛎、芒硝 | 单独存放、密封、芒硝需阴凉 |
| 芳香类密封 | 桂枝、麻黄、细辛、当归、川芎、吴茱萸 | 密封、防串味 |
| 胶类单独 | 阿胶、饴糖、麦冬 | 阴凉、夏季冷藏 |
| 鲜药单独 | 生姜 | 阴凉通风、定期更换 |
