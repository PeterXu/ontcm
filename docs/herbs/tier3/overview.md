## 必备要点

### 饴糖
- **小建中汤必备**，缺了方子不完整
- 温中补虚、缓急止痛
- 配桂枝+芍药 → 建中止痛（小建中汤）

### 芒硝
- **承气汤必备**，软坚攻下
- 配大黄 → 峻下热结（大承气汤）
- 注意：易吸潮融化、风化失水，需密封存放

### 粳米
- **白虎汤必备**，养胃生津
- 配石膏+知母 → 清热生津（白虎汤）
