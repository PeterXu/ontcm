## 六经归属速查

```
太阳主药：桂枝、麻黄
阳明主药：大黄、黄芩
少阳主药：柴胡、黄芩、半夏
太阴主药：白术、干姜、人参、茯苓
少阴主药：附子、干姜、人参
厥阴主药：当归、吴茱萸（第二档）

六经通用：甘草、大枣、生姜、芍药
```
