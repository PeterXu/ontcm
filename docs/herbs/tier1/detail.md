# 第一档药证详解

> 15味核心药材的完整药证、配伍要点、注意事项

---

## 甘草（调和之王）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 调和诸药 | 方中有寒热补泻之品 | 桂枝汤、半夏泻心汤 |
| 缓急止痛 | 腹痛、筋急 | 小建中汤、芍药甘草汤 |
| 补中益气 | 脾虚气弱 | 四君子汤类 |
| 祛痰止咳 | 咳嗽痰多 | 麻杏石甘汤 |

**配伍要点**：
- 甘草+桂枝 → 调和营卫（桂枝汤）
- 甘草+芍药 → 缓急止痛（芍药甘草汤）
- 甘草+干姜 → 温肺化饮（甘草干姜汤）

**注意**：大量长期使用可致水肿，中满者慎用

---

## 桂枝（太阳解表主药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 太阳表证 | 发热恶风、汗出脉浮缓 | 桂枝汤 |
| 太阳表实 | 无汗脉浮紧（配麻黄） | 麻黄汤 |
| 温通经脉 | 肢冷、痹痛 | 桂枝附子汤 |
| 平冲降逆 | 奔豚气 | 桂枝加桂汤 |
| 化气利水 | 小便不利 | 五苓散 |

**配伍要点**：
- 桂枝+麻黄 → 发汗解表（麻黄汤）
- 桂枝+芍药 → 调和营卫（桂枝汤）
- 桂枝+茯苓 → 化气利水（苓桂术甘汤）
- 桂枝+附子 → 温阳通脉（桂枝附子汤）

**剂量提示**：
- 解表：三两(9-15g)
- 平冲：五两(15-24g，桂枝加桂汤)

---

## 生姜（散寒止呕要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 散寒解表 | 恶寒发热 | 桂枝汤 |
| 和胃止呕 | 呕吐、恶心 | 小柴胡汤 |
| 温肺化饮 | 寒饮咳喘 | 小青龙汤 |
| 散水气 | 心下有水气 | 小青龙汤 |

**配伍要点**：
- 生姜+半夏 → 止呕要药对（小柴胡汤）
- 生姜+大枣 → 调和营卫（桂枝汤）
- 生姜+干姜 → 寒热并用（小青龙汤）

**注意**：切片用，不去皮

---

## 大枣（补中益气）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 补中益气 | 脾虚食少 | 桂枝汤、小柴胡汤 |
| 养血安神 | 血虚失眠 | 甘麦大枣汤 |
| 缓和药性 | 方中有峻烈之品 | 十枣汤、葶苈大枣泻肺汤 |

**配伍要点**：
- 大枣+生姜 → 调和营卫（桂枝汤）
- 大枣+甘草 → 补中益气（小建中汤）

**注意**：劈开用

---

## 芍药（养血敛阴）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 养血敛阴 | 表虚汗出、脉浮缓 | 桂枝汤 |
| 缓急止痛 | 腹痛、筋急 | 小建中汤、芍药甘草汤 |
| 利水 | 小便不利 | 真武汤 |
| 柔肝 | 肝气急 | 四逆散 |

**配伍要点**：
- 芍药+桂枝 → 调和营卫（桂枝汤）
- 芍药+甘草 → 缓急止痛（芍药甘草汤）
- 芍药+柴胡 → 柔肝疏肝（四逆散）

**白芍 vs赤芍**：
- 白芍：养血敛阴，用于虚证
- 赤芍：活血散瘀，用于实证

---

## 半夏（化痰止呕要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 化痰止呕 | 呕吐、痰多 | 小柴胡汤 |
| 消痞散结 | 心下痞满 | 半夏泻心汤 |
| 化饮止咳 | 寒饮咳喘 | 小青龙汤 |
| 和胃降逆 | 喜呕 | 小柴胡汤 |

**配伍要点**：
- 半夏+生姜 → 止呕要药对（小柴胡汤）
- 半夏+黄芩+黄连 → 寒热并用消痞（半夏泻心汤）
- 半夏+茯苓 → 化痰利水（小半夏加茯苓汤）

**注意**：法半夏为常用炮制品，生半夏有毒慎用

---

## 干姜（温中散寒要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 温中散寒 | 腹冷痛、呕吐 | 理中汤 |
| 回阳救逆 | 亡阳厥逆（配附子） | 四逆汤 |
| 温肺化饮 | 寒饮咳喘 | 小青龙汤 |
| 温经止血 | 脾不统血 | 甘草干姜汤 |

**配伍要点**：
- 干姜+附子 → 回阳救逆（四逆汤）
- 干姜+人参+白术 → 温中健脾（理中汤）
- 干姜+细辛+五味子 → 温肺化饮（小青龙汤）

**干姜 vs生姜**：
- 干姜：温中散寒，守而不走
- 生姜：散寒解表，走而不守

---

## 人参（大补元气）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 大补元气 | 气虚欲脱 | 理中汤 |
| 健脾益气 | 脾虚食少、腹胀 | 理中汤、四君子汤 |
| 生津止渴 | 热病津伤 | 白虎加人参汤 |
| 安神定志 | 心悸失眠 | 人参养荣汤 |

**配伍要点**：
- 人参+白术+干姜 → 温中健脾（理中汤）
- 人参+石膏 → 清热生津（白虎加人参汤）
- 人参+附子 → 回阳固脱（四逆加人参汤）

**人参 vs党参**：
- 人参：力强，用于急证、重证
- 党参：力弱，用于日常调理

**经方用量**：二两至三两(6-9g)，可用党参替代

---

## 附子（回阳救逆要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 回阳救逆 | 亡阳厥逆、脉微 | 四逆汤 |
| 温阳利水 | 水气凌心 | 真武汤 |
| 温经散寒 | 寒痹疼痛 | 桂枝附子汤 |
| 补火助阳 | 阳虚畏寒 | 附子汤 |

**配伍要点**：
- 附子+干姜 → 回阳救逆（四逆汤）
- 附子+茯苓+白术 → 温阳利水（真武汤）
- 附子+桂枝 → 温经散寒（桂枝附子汤）

**毒性管理**：
- 炮附子：毒性降低，常用
- 生附子：毒性大，慎用
- 久煎：减毒（先煎30-60分钟）
- 剂量：一枚(10-15g)为经方常用量

**注意**：毒性饮片，专用处方、双人复核、限量使用

---

## 茯苓（利水渗湿要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 利水渗湿 | 小便不利、水肿 | 五苓散 |
| 健脾宁心 | 心悸失眠 | 苓桂术甘汤 |
| 化痰饮 | 饮心下悸 | 苓桂甘枣汤 |
| 利水健脾 | 脾虚水肿 | 真武汤 |

**配伍要点**：
- 茯苓+桂枝 → 化气利水（苓桂术甘汤）
- 茯苓+白术 → 健脾利水（五苓散）
- 茯苓+猪苓+泽泻 → 利水消肿（五苓散）

---

## 大黄（阳明攻下主药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 攻下积滞 | 不大便、腹胀痛 | 大承气汤 |
| 泻热通便 | 热结便秘 | 大承气汤 |
| 活血祛瘀 | 瘀血证（配桃仁） | 桃核承气汤 |
| 清热泻火 | 实热证 | 大黄黄连泻心汤 |

**配伍要点**：
- 大黄+芒硝 → 峻下热结（大承气汤）
- 大黄+厚朴+枳实 → 行气攻下（小承气汤）
- 大黄+桃仁 → 活血祛瘀（桃核承气汤）

**剂量提示**：
- 攻下：四两(12g)
- 活血：二两(6g)
- 清热：二两(6g)

**注意**：后下，久煎泻下力减弱

---

## 麻黄（太阳解表主药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 发汗解表 | 无汗、恶寒、脉浮紧 | 麻黄汤 |
| 平喘止咳 | 咳喘 | 麻杏石甘汤 |
| 利水消肿 | 水肿 | 越婢汤 |
| 散寒除痹 | 风寒湿痹 | 麻黄加术汤 |

**配伍要点**：
- 麻黄+桂枝 → 发汗解表（麻黄汤）
- 麻黄+杏仁 → 止咳平喘（麻杏石甘汤）
- 麻黄+石膏 → 清热平喘（麻杏石甘汤）
- 麻黄+附子+细辛 → 温阳解表（麻黄细辛附子汤）

**注意**：
- 体虚、高血压、心脏病慎用
- 剂量：二至三两(6-9g)

---

## 白术（太阴健脾主药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 健脾燥湿 | 脾虚湿盛、腹胀 | 理中汤 |
| 利水消肿 | 小便不利 | 五苓散 |
| 固表止汗 | 表虚汗出 | 玉屏风散 |
| 安胎 | 胎动不安 | 当归散 |

**配伍要点**：
- 白术+人参+干姜 → 温中健脾（理中汤）
- 白术+茯苓+桂枝 → 利水健脾（五苓散）
- 白术+附子 → 温阳利水（真武汤）

**白术 vs苍术**：
- 白术：健脾益气，用于脾虚
- 苍术：燥湿力强，用于湿盛

---

## 黄芩（清热燥湿要药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 清热燥湿 | 湿热证 | 半夏泻心汤 |
| 泻火解毒 | 实热证 | 黄连解毒汤 |
| 清少阳热 | 往来寒热 | 小柴胡汤 |
| 清肺热 | 肺热咳喘 | 麻杏石甘汤（黄芩可替代） |

**配伍要点**：
- 黄芩+柴胡 → 和解少阳（小柴胡汤）
- 黄芩+黄连+半夏 → 寒热并用消痞（半夏泻心汤）
- 黄芩+芍药 → 清热养血（黄芩汤）

---

## 柴胡（少阳和解主药）

| 药证 | 临床表现 | 方剂举例 |
|------|----------|----------|
| 和解少阳 | 往来寒热、胸胁苦满 | 小柴胡汤 |
| 疏肝解郁 | 肝郁气滞 | 四逆散 |
| 升举阳气 | 气虚下陷 | 补中益气汤 |
| 解表退热 | 发热 | 柴胡桂枝汤 |

**配伍要点**：
- 柴胡+黄芩 → 和解少阳（小柴胡汤）
- 柴胡+芍药 → 疏肝柔肝（四逆散）
- 柴胡+桂枝 → 和解兼解表（柴胡桂枝汤）

**剂量提示**：
- 和解少阳：半斤(15-24g)
- 疏肝：四至五钱(12-15g)
- 升阳：二至三钱(6-9g)
