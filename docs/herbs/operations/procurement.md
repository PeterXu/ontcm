## 进货要点

| 类别 | 药味 | 备注 |
|------|------|------|
| 消耗大户 | 甘草、生姜、大枣 | 月用量>2kg，多备 |
| 频率低但量大 | 石膏 | 单次30-60g，备2包 |
| 用量小但必备 | 黄连、细辛、吴茱萸 | 1包能用数月，但不可缺 |
| 毒性饮片 | 附子 | 专用处方、双人复核 |

## 进货优先级

### 第一阶段：必进15味（1240元）
```
甘草 桂枝 生姜 大枣 芍药 半夏 干姜 人参 附子 茯苓 大黄 麻黄 白术 黄芩 柴胡
```
- 覆盖近90%用药频次
- 可开出大部分经方
- 预算：1240元

### 第二阶段：补充28味（1500元）
```
黄连 细辛 枳实 厚朴 石膏 当归 杏仁 五味子 葛根 龙骨 牡蛎 栀子 泽泻 猪苓
阿胶 麦冬 桃仁 牡丹皮 川芎 吴茱萸 黄芪 山药 山茱萸 熟地黄 天花粉 苏梗 连翘 桔梗
```
- 让经方体系完整
- 可开出全部经方
- 预算：1500元

### 第三阶段：按需10味（300-500元）
```
饴糖 粳米 芒硝 葶苈子 旋覆花 代赭石 赤石脂 升麻 防己 薏苡仁
```
- 特殊方剂专用
- 按需求逐步补充
- 预算：300-500元

## 进货渠道建议

| 药材类型 | 渠道建议 | 注意事项 |
|----------|----------|----------|
| 常用药材 | 当地药材公司、GMP认证企业 | 质量稳定、货源充足 |
| 毒性饮片 | 专项采购、资质齐全企业 | 专用处方、双人复核 |
| 矿物类 | 正规药材公司 | 注意质量、避免杂质 |
| 胶类（阿胶） | 大品牌（东阿阿胶等） | 注意真伪鉴别 |
| 饴糖 | 当地食材市场或药材公司 | 易变质、不宜久存 |
| 生姜 | 当地菜市场 | 鲜品、定期更换 |
