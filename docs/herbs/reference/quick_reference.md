## 二、六经主药速查

```
太阳：桂枝、麻黄

少阳：柴胡、黄芩、半夏
太阴：白术、干姜、人参、茯苓
少阴：附子、干姜
厥阴：当归、吴茱萸
```

## 六、辨证用药速查

| 症状 | 常用药 | 备注 |
|------|--------|------|
| 发热恶风汗出 | 桂枝、芍药 | 太阳表虚 |
| 无汗恶寒脉紧 | 麻黄、桂枝 | 太阳表实 |
| 往来寒热胸胁满 | 柴胡、黄芩 | 少阳证 |
| 腹满吐利不渴 | 白术、干姜、人参 | 太阴脾虚寒 |
| 但欲寐肢冷脉微 | 附子、干姜 | 少阴亡阳 |
| 不大便腹胀痛谵语 | 大黄、芒硝 | 阳明腑实 |
| 大热大汗大渴脉洪 | 石膏、知母 | 阳明经热 |
| 呕吐痰多 | 半夏、生姜 | 化痰止呕 |
| 小便不利水肿 | 茯苓、泽泻、猪苓 | 利水渗湿 |
| 心悸失眠 | 茯苓、龙骨、牡蛎 | 安神宁心 |
| 腹痛筋急 | 芍药、甘草 | 缓急止痛 |
