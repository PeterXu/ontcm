# 承气汤类药证详解

> 承气汤类为阳明腑实主方，攻下通腑、泻热存阴。

---

## 一、《伤寒论》原文

> "阳明病，谵语，有潮热，反不能食者，胃中必有燥屎五六枚也；若能食者，但硬尔。宜大承气汤下之。"（215条）

> "伤寒若吐若下后不解，不大便五六日，上至十余日，日晡所发潮热，不恶寒，独语如见鬼状；若剧者，发则不识人，循衣摸床，惕而不安，微喘直视，脉弦者生，涩者死。微者，但发热谵语者，大承气汤主之。"（212条）

---

## 二、方剂组成（三方对比）

### 大承气汤

| 药味 | 剂量（原方） | 功效 |
|------|------------|------|
| 大黄 | 四两（酒洗） | 攻下泻热 |
| 厚朴 | 半斤 | 行气除满 |
| 枳实 | 五枚 | 破气消积 |
| 芒硝 | 三合 | 软坚泻下 |

### 小承气汤

| 药味 | 剂量（原方） | 功效 |
|------|------------|------|
| 大黄 | 四两 | 攻下泻热 |
| 厚朴 | 二两 | 行气除满 |
| 枳实 | 三枚 | 破气消积 |

**特点**：无芒硝，泻下力较轻。

### 调胃承气汤

| 药味 | 剂量（原方） | 功效 |
|------|------------|------|
| 大黄 | 四两（酒洗） | 攻下泻热 |
| 芒硝 | 半升 | 软坚泻下 |
| 甘草 | 二两 | 调和护胃 |

**特点**：无厚朴枳实，调和胃气，泻下最缓。

---

## 三、方证要点

### 核心病机

阳明腑实（有形）——燥屎内结，腑气不通，热盛伤津。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 日晡潮热 | 下午3-5点发热明显 | 阳明经气旺时热甚 |
| 腹满痛拒按 | 腹胀痛明显，拒按 | 燥屎内结 |
| 大便不通 | 便秘或热结旁流 | 腑气不通 |
| 谵语 | 神志不清、胡言乱语 | 热扰心神 |
| 苔焦黑起刺 | 舌苔焦黑、起刺芒 | 热极津枯 |
| 脉沉实 | 脉沉而有力 | 里实热 |

### 三方使用鉴别

| 对比项 | 大承气汤 | 小承气汤 | 调胃承气汤 |
|-------|---------|---------|-----------|
| **痞（腹满）** | 重 | 轻 | 无或轻 |
| **满（腹胀）** | 重 | 中 | 无或轻 |
| **燥（便干）** | 重 | 轻 | 中 |
| **实（拒按）** | 重 | 轻 | 无 |
| **谵语** | 可有 | 无或轻 | 可有心烦 |
| **脉象** | 沉实 | 沉滑 | 沉 |
| **芒硝** | 有 | 无 | 有 |
| **厚朴枳实** | 有量重 | 有量轻 | 无 |
| **甘草** | 无 | 无 | 有 |

**使用原则**：
- 痞满燥实俱全 → 大承气汤
- 痞满为主、燥实轻 → 小承气汤
- 燥为主、痞满轻 → 调胃承气汤

## 四、药证校验

### 大黄——攻下泻热

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 攻下 | 大便不通 | 燥屎内结 ✓ |
| 泻热 | 发热、谵语 | 热盛 ✓ |

**大黄药证**：凡见便秘、发热者，大黄有据。

### 芒硝——软坚泻下

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 软坚 | 大便干硬 | 燥屎 ✓ |
| 泻下 | 大便不通 | 配合大黄 ✓ |

**芒硝药证**：凡见大便干硬难下者，芒硝有据。

### 厚朴——行气除满

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 行气 | 腹胀明显 | 腑气不通 ✓ |
| 除满 | 腹满 | 气滞 ✓ |

**厚朴药证**：凡见腹胀明显者，厚朴有据。

### 枳实——破气消积

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 破气 | 腹胀痛拒按 | 气滞实证 ✓ |
| 消积 | 大便不通 | 配合厚朴 ✓ |

**枳实药证**：凡见腹胀痛拒按者，枳实有据。

---

## 五、禁忌 ⚠️

| 禁忌人群 | 原因 | 替代方案 |
|---------|------|---------|
| 孕妇 | 攻下伤胎，可能流产 | 改用润下或缓下 |
| 体虚者、老人 | 攻下伤正，正气更虚 | 改用增液承气或润下 |
| 产后、经期 | 攻下伤气血 | 改用温和通便法 |
| 表证未解 | 攻下引邪入里 | 先解表再攻下 |
| 无腑实表现 | 方证不符 | 需辨证 |

## 六、典型案例

### 案例：阳明腑实（大承气汤证）

**患者**：男，50岁

**主诉**：便秘、发热5天

**十问**：
- 吃：不能吃
- 喝：口渴
- 拉：便秘，5天未大便
- 撒：尿少色黄
- 睡：谵语，神志不清
- 汗：无大汗
- 痛：腹痛拒按
- 寒热：下午发热明显（日晡潮热）

**舌脉**：苔焦黑起刺，脉沉实

**辨证**：阳明腑实

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 日晡潮热 | 下午发热 ✓ | ✓ |
| 腹满痛拒按 | 腹痛拒按 ✓ | ✓ |
| 大便不通 | 5天未便 ✓ | ✓ |
| 谵语 | 神志不清 ✓ | ✓ |
| 苔焦黑 | 苔焦黑 ✓ | ✓ |
| 脉沉实 | 脉沉实 ✓ | ✓ |

**处方**：大承气汤
- 大黄 12g（后下）
- 厚朴 15g
- 枳实 12g
- 芒硝 9g（溶服）

**医嘱**：得下即止，不可过服。

## 七、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 日晡潮热+腹满痛拒按+便秘+谵语+苔焦黑+脉沉实 |
| **核心病机** | 阳明腑实（燥屎内结） |
| **三方选择** | 痞满燥实俱全→大承气；痞满为主→小承气；燥为主→调胃承气 |
| **禁忌** | 孕妇、体虚老人、产后经期禁用 |
| **鉴别** | 与白虎汤鉴别：有无腑实 |

**记住**：承气汤攻下力强，用前必排查禁忌！孕妇、体虚者禁用。得下即止，不可过服。
