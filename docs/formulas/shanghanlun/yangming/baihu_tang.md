# 白虎汤药证详解

> 白虎汤为阳明经热主方，清热生津。

---

## 一、《伤寒论》原文

> "伤寒脉浮滑，此以表有热，里有寒，白虎汤主之。"（176条）

> "三阳合病，腹满身重，难以转侧，口不仁面垢，谵语遗尿，发汗则谵语甚，下之则额上生汗，手足逆冷。若自汗出者，白虎汤主之。"（219条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 石膏 | 一斤 | 清热泻火 | 肺、胃 |
| 知母 | 六两 | 清热滋阴 | 肺、胃、肾 |
| 甘草（炙） | 二两 | 调和诸药 | 心、肺、脾 |
| 粳米 | 六合 | 护胃生津 | 脾、胃 |

**煮服法**：上四味，以水一斗，煮米熟汤成，去滓，温服一升，日三服。

---

## 三、方证要点

### 核心病机

阳明经热（无形）——热盛伤津，弥漫全身，尚未腑实。

### 方证对照表（"四大"）

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 大热 | 高热 | 阳明热盛 |
| 大汗 | 大汗出 | 热迫津液外泄 |
| 大渴 | 口渴明显、喜冷饮 | 热盛伤津 |
| 脉洪大 | 脉洪大有力 | 热盛正气未虚 |
| 舌红苔黄燥 | 舌红、苔黄燥 | 热盛伤津 |
| 无腑实 | 无腹满痛拒按、无便秘 | 热未入腑 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| ≥4条（含四大） | 高 | 方证匹配，可用 |
| 有腑实（腹满痛便秘） | — | 改用承气汤 |
| 表证未解 | — | 先解表再清里 |

---

## 四、药证校验

### 石膏——清热泻火

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 清热 | 大热 | 阳明热盛 ✓ |
| 止渴 | 大渴 | 热盛伤津 ✓ |

**石膏药证**：凡见高热、口渴者，石膏有据。

### 知母——清热滋阴

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 清热 | 大热 | 配合石膏 ✓ |
| 滋阴 | 大渴、苔燥 | 热伤津 ✓ |

**知母药证**：凡见高热、口渴、苔燥者，知母有据。

### 粳米——护胃生津

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 护胃 | — | 防石膏知母寒凉伤胃 ✓ |
| 生津 | 大渴 | 配合知母 ✓ |

### 炙甘草——调和诸药

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 调和 | — | 使药力缓和 ✓ |

---

## 五、加减法

| 加减方 | 组成变化 | 适用 |
|-------|---------|------|
| 白虎加人参汤 | +人参 | 汗多、脉大无力（气津两伤） |
| 白虎加桂枝汤 | +桂枝 | 温疟（热+表证） |
| 白虎加苍术汤 | +苍术 | 湿温（热+湿） |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 表证未解 | 清里过早引邪入里 | 先解表，表解再清里 |
| 无汗（非阳明经热特征） | 方证可能不符 | 需辨证是否阳明 |
| 腑实（腹满痛便秘） | 有形之热，需攻下 | 改用承气汤 |
| 脉细弱（气津大伤） | 正气已虚 | 改用白虎加人参汤 |

---

## 七、典型案例

### 案例：阳明经热

**患者**：男，40岁

**主诉**：高热2天

**十问**：
- 吃：不想吃
- 喝：口渴明显，想喝凉水，喝得多
- 拉：正常
- 撒：尿少色黄
- 睡：睡不着
- 汗：大汗出
- 痛：无腹痛
- 寒热：高热，不怕冷

**舌脉**：舌红苔黄燥，脉洪大

**辨证**：阳明经热

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 大热 | 高热 ✓ | ✓ |
| 大汗 | 大汗出 ✓ | ✓ |
| 大渴 | 口渴明显喜凉 ✓ | ✓ |
| 脉洪大 | 脉洪大 ✓ | ✓ |
| 舌红苔黄燥 | 舌红苔黄燥 ✓ | ✓ |
| 无腑实 | 无腹痛便秘 ✓ | ✓ |

**处方**：白虎汤
- 石膏 30g
- 知母 9g
- 炙甘草 6g
- 粳米 15g

## 八、方剂对比

### 白虎汤 vs 承气汤

| 对比项 | 白虎汤 | 承气汤 |
|-------|-------|-------|
| 病机 | 阳明经热（无形） | 阳明腑实（有形） |
| 核心症状 | 大热大汗大渴脉洪大 | 腹满痛拒按+便秘+谵语 |
| 腹部 | 无腹胀痛 | 腹满痛拒按 ✓ |
| 大便 | 可能正常 | 便秘或热结旁流 ✓ |
| 舌象 | 苔黄燥 | 苔焦黑起刺 |
| 神志 | 清楚 | 可有谵语 |

**鉴别要点**：有无腑实（腹满痛拒按、便秘）。

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 大热+大汗+大渴+脉洪大（四大） |
| **核心病机** | 阳明经热（无形之热） |
| **核心药证** | 石膏→高热；知母→口渴、苔燥 |
| **禁忌** | 表证未解先解表；有腑实改承气汤 |
| **鉴别** | 与承气汤鉴别关键：有无腑实 |

**记住**：白虎汤治阳明经热，关键是"四大"。有腹满痛便秘者改用承气汤。
