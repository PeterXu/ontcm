# 麻子仁丸药证详解

> 麻子仁丸治脾约，润肠通便。

---

## 一、《伤寒论》原文

> "趺阳脉浮而涩，浮则胃气强，涩则小便数，浮涩相搏，大便则硬，其脾为约，麻子仁丸主之。"（247条）

---

## 二、方剂组成

| 药味 | 功效 |
|------|------|
| 麻子仁 | 润肠通便 |
| 芍药 | 敛阴 |
| 枳实 | 破气 |
| 大黄 | 通便 |
| 厚朴 | 行气 |
| 杏仁 | 润肠 |

---

## 三、方证要点

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 大便硬 | 便秘 | 脾约 |
| 小便数 | 尿频 | 津偏渗膀胱 |

---

## 四、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 大便硬+小便数 |
| **核心病机** | 脾约（胃肠燥热） |
| **特点** | 润肠通便 |

**记住**：麻子仁丸治脾约便秘。
