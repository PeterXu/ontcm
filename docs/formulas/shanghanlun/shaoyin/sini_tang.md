# 四逆汤药证详解

> 四逆汤为少阴病主方，回阳救逆、温里散寒。

---

## 一、《伤寒论》原文

> "少阴病，脉微细，但欲寐也。"（281条）

> "少阴病，脉沉者，急温之，宜四逆汤。"（323条）

> "少阴病，下利清谷，里寒外热，手足厥逆，脉微欲绝，身反不恶寒，其人面色赤，或腹痛，或干呕，或咽痛，或利止脉不出者，通脉四逆汤主之。"（317条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 附子 | 一枚（生用，去皮，破八片） | 回阳救逆、温里散寒 | 心、肾、脾 |
| 干姜 | 一两半 | 温中散寒、回阳 | 脾、胃、心、肺 |
| 甘草（炙） | 二两 | 调和诸药、缓附子毒性 | 心、肺、脾 |

**煮服法**：上三味，以水三升，煮取一升二合，去滓，分温再服。强人可大附子一枚、干姜三两。

---

## 三、方证要点

### 核心病机

少阴阳虚寒盛——阳气衰微，阴寒内盛，阳气欲脱。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 但欲寐 | 精神萎靡、嗜睡 | 心肾阳虚，神明失养 |
| 四肢厥逆 | 手足冰冷 | 阳气不能达于四末 |
| 下利清谷 | 腹泻、完谷不化 | 脾肾阳虚，腐熟无能 |
| 脉微细 | 脉极细极软 | 阳气虚衰，血脉不充 |
| 或脉微欲绝 | 脉微弱至极 | 亡阳危候 |
| 恶寒蜷卧 | 怕冷、蜷缩卧 | 阴寒内盛 |
| 冷汗淋漓 | 冷汗不止 | 亡阳危候 ⚠️ |

### 少阴危候识别 ⚠️

| 危候表现 | 说明 | 处理 |
|---------|------|------|
| 但欲寐+肢冷+脉微欲绝 | 少阴亡阳 | 急用四逆汤抢救+送医 |
| 冷汗淋漓+脉微 | 亡阳危候 | 急救 |
| 下利清谷不止 | 阳气将脱 | 急温 |

---

## 四、药证校验

### 附子——回阳救逆

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 回阳 | 肢冷、脉微 | 阳气衰微 ✓ |
| 救逆 | 但欲寐、神萎 | 心阳虚 ✓ |
| 温里 | 下利清谷 | 脾肾阳虚 ✓ |
| 散寒 | 恶寒蜷卧 | 阴寒内盛 ✓ |

**附子药证**：凡见肢冷、脉微、但欲寐者，附子有据。

**⚠️ 附子有毒，需久煎（1小时以上），先煎减毒。**

### 干姜——温中散寒

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温中 | 下利清谷 | 脾阳虚 ✓ |
| 散寒 | 恶寒、肢冷 | 寒盛 ✓ |
| 回阳 | 脉微 | 配合附子 ✓ |

**干姜药证**：凡见下利清谷、恶寒者，干姜有据。

### 炙甘草——调和缓毒

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 调和诸药 | — | 使附子干姜力缓 ✓ |
| 缓附子毒性 | — | 减附子毒性 ✓ |

---

## 五、加减法

| 加减方 | 组成变化 | 适用 |
|-------|---------|------|
| 通脉四逆汤 | 附子大者一枚、干姜三两、甘草二两 | 脉微欲绝、里寒外热（假热真寒） |
| 四逆加人参汤 | +人参 | 气阴两伤、亡阳亡阴 |
| 白通汤 | 去甘草，加葱白 | 阴盛格阳、下利 |
| 白通加猪胆汁汤 | +猪胆汁 | 阴盛格阳、格拒不下 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 真热假寒（内热外寒） | 热证用热药加重病情 | 需先辨寒热真假 |
| 阴虚发热 | 温药加重阴虚 | 改用养阴清热方 |
| 阳明热盛 | 方证不符 | 改用清热方 |

**⚠️ 寒热真假鉴别要点**：
| 鉴别点 | 真寒证 | 真热假寒 |
|-------|-------|---------|
| 喜冷/喜热 | 喜热 | 喜冷（虽肢冷） |
| 尿色 | 清 | 黄 |
| 舌象 | 淡 | 红 |
| 脉象 | 微细无力 | 数而有力 |

## 七、典型案例

### 案例：少阴亡阳

**患者**：男，65岁

**主诉**：腹泻、精神萎靡2天

**十问**：
- 吃：不想吃
- 喝：不渴
- 拉：腹泻清谷，日十余次
- 撒：尿清
- 睡：嗜睡，精神萎靡（但欲寐）
- 汗：冷汗出
- 痛：无腹痛
- 寒热：怕冷，四肢冰冷

**舌脉**：舌淡苔白滑，脉微细欲绝

**辨证**：少阴亡阳

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 但欲寐 | 嗜睡 ✓ | ✓ |
| 四肢厥逆 | 四肢冰冷 ✓ | ✓ |
| 下利清谷 | 腹泻清谷 ✓ | ✓ |
| 脉微细欲绝 | 脉微细欲绝 ✓ | ✓ |
| 冷汗 | 冷汗出 ✓ | ✓ |

**处置**：
- 急煎四逆汤
- 同时送医监护

**处方**：四逆汤
- 附子 15g（先煎1小时）
- 干姜 10g
- 炙甘草 10g

## 八、方剂对比

### 四逆汤 vs 理中汤

| 对比项 | 四逆汤 | 理中汤 |
|-------|-------|-------|
| 病位 | 下焦（肾） | 中焦（脾） |
| 病机 | 肾阳虚衰 | 脾虚寒湿 |
| 核心症状 | 但欲寐、肢冷、脉微 | 腹满吐利 |
| 精神状态 | 精神萎靡 ✓ | 神志清楚 |
| 脉象 | 微细欲绝 ✓ | 沉弱 |
| 病势 | 急，危候 | 缓 |
| 主药 | 附子 | 人参、白术 |

**鉴别要点**：精神状态、脉象强弱、病势缓急。

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 但欲寐+肢冷+下利清谷+脉微细+冷汗 |
| **核心病机** | 少阴阳虚寒盛（亡阳） |
| **核心药证** | 附子→肢冷脉微；干姜→下利清谷 |
| **禁忌** | 真热假寒、阴虚发热禁用 |
| **煎法** | 附子需先煎1小时以上减毒 |
| **危急** | 少阴亡阳需急救+送医 |

**记住**：四逆汤是少阴亡阳急救方，用前必辨寒热真假！附子有毒，必久煎。见危候需送医。
