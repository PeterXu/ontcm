# 真武汤药证详解

> 真武汤为少阴水气主方，温阳利水。

---

## 一、《伤寒论》原文

> "太阳病发汗，汗出不解，其人仍发热，心下悸，头眩，身瞤动，振振欲擗地者，真武汤主之。"（82条）

> "少阴病，二三日不已，至四五日，腹痛，小便不利，四肢沉重疼痛，自下利者，此为有水气。其人或咳，或小便不利，或下利，或呕者，真武汤主之。"（316条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 附子 | 一枚（炮去皮） | 温阳利水 | 心、肾、脾 |
| 白术 | 二两 | 健脾燥湿 | 脾、胃 |
| 茯苓 | 三两 | 利水渗湿 | 心、脾、肾 |
| 生姜 | 三两 | 散水气 | 肺、脾、胃 |
| 芍药 | 三两 | 利小便、缓急 | 肝、脾 |

**煮服法**：上五味，以水八升，煮取三升，去滓，温服七合，日三服。

---

## 三、方证要点

### 核心病机

少阴水气——肾阳虚衰，水气内停，泛溢肌肤。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 小便不利 | 尿少 | 阳虚气化不利 |
| 四肢沉重疼痛 | 四肢肿重疼痛 | 水气泛溢 |
| 腹痛 | 腹痛 | 寒凝气滞 |
| 下利 | 腹泻 | 水湿下注 |
| 或咳 | 咳嗽 | 水气凌肺 |
| 或呕 | 恶心呕吐 | 水气犯胃 |
| 心下悸 | 心悸 | 水气凌心 |
| 头眩 | 头晕目眩 | 水气上扰 |
| 身瞤动 | 身体抽动 | 阳虚水泛 |
| 苔白滑 | 舌苔白滑 | 水湿内停 |
| 脉沉 | 脉沉 | 里虚寒 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 水气征象+阳虚征象 | 高 | 方证匹配，可用 |
| 无水气征象 | — | 改用四逆汤 |

---

## 四、药证校验

### 附子——温阳利水

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温阳 | 四肢沉重、脉沉 | 阳虚 ✓ |
| 利水 | 小便不利 | 气化不利 ✓ |

**附子药证**：凡见阳虚+水气者，附子有据。

### 茯苓——利水渗湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 利水 | 小便不利 | 水气内停 ✓ |
| 宁心 | 心下悸 | 水气凌心 ✓ |

**茯苓药证**：凡见小便不利、心悸者，茯苓有据。

### 白术——健脾燥湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 健脾 | 下利 | 脾虚湿盛 ✓ |
| 燥湿 | 四肢沉重 | 水湿内停 ✓ |

### 生姜——散水气

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 散水 | 呕 | 水气犯胃 ✓ |
| 和胃 | 呕 | 胃气上逆 ✓ |

### 芍药——利小便、缓急

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 利小便 | 小便不利 | 助茯苓利水 ✓ |
| 缓急 | 腹痛 | 筋脉拘急 ✓ |

---

## 五、加减法

| 或然证 | 加减 | 依据 |
|-------|------|------|
| 咳 | 加五味子、细辛、干姜 | 温肺止咳 |
| 小便利 | 去茯苓 | 无需利水 |
| 下利 | 去芍药，加干姜 | 温中止泻 |
| 呕 | 去附子，加生姜 | 重用生姜止呕 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 阴虚水肿 | 温阳利水伤阴 | 改用滋阴利水方 |
| 热证水肿 | 热证用热药 | 改用清热利水方 |

---

## 七、典型案例

### 案例：少阴水气

**患者**：男，60岁

**主诉**：水肿、心悸1周

**十问**：
- 吃：不想吃
- 喝：不渴
- 拉：腹泻
- 撒：尿少
- 睡：心悸睡不着
- 汗：无
- 痛：腹痛、四肢沉重
- 寒热：怕冷

**舌脉**：舌淡胖苔白滑，脉沉

**辨证**：少阴水气

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 小便不利 | 尿少 ✓ | ✓ |
| 四肢沉重疼痛 | 四肢沉重 ✓ | ✓ |
| 腹痛 | 腹痛 ✓ | ✓ |
| 下利 | 腹泻 ✓ | ✓ |
| 心下悸 | 心悸 ✓ | ✓ |
| 苔白滑 | 苔白滑 ✓ | ✓ |
| 脉沉 | 脉沉 ✓ | ✓ |

**处方**：真武汤
- 附子 10g（先煎）
- 白术 10g
- 茯苓 15g
- 生姜 10g
- 芍药 10g

## 八、方剂对比

### 真武汤 vs 四逆汤

| 对比项 | 真武汤 | 四逆汤 |
|-------|-------|-------|
| 病机 | 少阴水气 | 少阴亡阳 |
| 核心症状 | 水肿+小便不利 | 但欲寐+肢冷 |
| 茯苓白术 | 有 | 无 |
| 生姜 | 有（散水） | 无 |
| 干姜 | 无 | 有 |

**鉴别要点**：有无水气征象（水肿、小便不利）。

### 真武汤 vs 五苓散

| 对比项 | 真武汤 | 五苓散 |
|-------|-------|-------|
| 病位 | 少阴（肾） | 太阳（膀胱） |
| 病机 | 阳虚水停 | 气化不利 |
| 附子 | 有 | 无 |
| 肉桂 | 无 | 有 |
| 泽泻猪苓 | 无 | 有 |

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 小便不利+四肢沉重疼痛+腹痛+下利+心悸+苔白滑 |
| **核心病机** | 少阴水气（阳虚水停） |
| **核心药证** | 附子→阳虚；茯苓→小便不利；白术→下利 |
| **特点** | 温阳+利水 |
| **禁忌** | 阴虚水肿、热证水肿禁用 |

**记住**：真武汤治少阴水气，关键是"阳虚+水气"——水肿、小便不利、四肢沉重。
