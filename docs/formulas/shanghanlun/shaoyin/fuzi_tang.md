# 附子汤药证详解

> 附子汤为少阴阳虚寒凝主方，温经扶阳、除湿止痛。

---

## 一、《伤寒论》原文

> "少阴病，得之一二日，口中和，其背恶寒者，当灸之，附子汤主之。"（304条）

> "少阴病，身体痛，手足寒，骨节痛，脉沉者，附子汤主之。"（305条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 附子 | 二枚（炮去皮） | 温阳散寒 | 心、肾、脾 |
| 茯苓 | 三两 | 利水渗湿 | 心、脾、肾 |
| 人参 | 二两 | 补气扶正 | 脾、肺 |
| 白术 | 四两 | 健脾燥湿 | 脾、胃 |
| 芍药 | 三两 | 和营止痛 | 肝、脾 |

**煮服法**：上五味，以水八升，煮取三升，去滓，温服一升，日三服。

---

## 三、方证要点

### 核心病机

少阴阳虚寒凝——肾阳虚衰，寒湿凝滞经脉。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 背恶寒 | 背部怕冷 | 阳虚寒凝 |
| 身体痛 | 身体疼痛 | 寒凝经脉 |
| 手足寒 | 手足冰冷 | 阳虚不达四末 |
| 骨节痛 | 关节疼痛 | 寒凝关节 |
| 口中和 | 口不苦不渴 | 无热 |
| 脉沉 | 脉沉 | 里寒 |
| 或下利 | 腹泻 | 寒湿下注 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 背恶寒+身痛+脉沉 | 高 | 方证匹配，可用 |
| 口渴口苦 | — | 有热象，考虑他方 |

---

## 四、药证校验

### 附子——温阳散寒

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温阳 | 背恶寒、手足寒 | 阳虚 ✓ |
| 散寒 | 身体痛、骨节痛 | 寒凝 ✓ |

**附子药证**：凡见背恶寒、身痛者，附子有据。

### 白术——健脾燥湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 燥湿 | 身痛、骨节痛 | 寒湿 ✓ |
| 健脾 | 下利（或然） | 脾虚 ✓ |

### 茯苓——利水渗湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 渗湿 | 身痛、骨节痛 | 寒湿 ✓ |

### 人参——补气扶正

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 扶正 | 乏力、脉沉 | 正气虚 ✓ |

### 芍药——和营止痛

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 止痛 | 身痛、骨节痛 | 寒凝致痛 ✓ |

---

## 五、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 口渴口苦 | 有热象 | 改用清热方 |
| 阴虚发热 | 温阳伤阴 | 改用养阴方 |

---

## 六、典型案例

### 案例：少阴阳虚寒凝

**患者**：男，55岁

**主诉**：背部怕冷、身体疼痛1周

**十问**：
- 吃：正常
- 喝：口中和
- 拉：正常
- 撒：正常
- 睡：正常
- 汗：无
- 痛：身体痛、关节痛
- 寒热：背部怕冷明显，手足冷

**舌脉**：舌淡苔白，脉沉

**辨证**：少阴阳虚寒凝

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 背恶寒 | 背部怕冷 ✓ | ✓ |
| 身体痛 | 身体疼痛 ✓ | ✓ |
| 手足寒 | 手足冷 ✓ | ✓ |
| 骨节痛 | 关节痛 ✓ | ✓ |
| 口中和 | 口中和 ✓ | ✓ |
| 脉沉 | 脉沉 ✓ | ✓ |

**处方**：附子汤
- 附子 15g（先煎）
- 茯苓 12g
- 人参 9g
- 白术 15g
- 芍药 12g

## 七、方剂对比

### 附子汤 vs 四逆汤

| 对比项 | 附子汤 | 四逆汤 |
|-------|-------|-------|
| 核心症状 | 背恶寒+身痛+骨节痛 | 但欲寐+肢冷+下利清谷 |
| 身痛 | 有 | 无 |
| 茯苓白术人参芍药 | 有 | 无 |
| 干姜 | 无 | 有 |

**鉴别要点**：有无身痛骨节痛（寒凝经脉）。

### 附子汤 vs 真武汤

| 对比项 | 附子汤 | 真武汤 |
|-------|-------|-------|
| 核心症状 | 身痛+骨节痛+背恶寒 | 小便不利+四肢沉重 |
| 人参 | 有 | 无 |
| 生姜 | 无 | 有 |
| 白术量 | 四两 | 二两 |

**鉴别要点**：痛（寒凝经脉）vs 水（水气内停）。

---

## 八、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 背恶寒+身体痛+手足寒+骨节痛+脉沉 |
| **核心病机** | 少阴阳虚寒凝 |
| **核心药证** | 附子→阳虚寒凝；白术茯苓→寒湿；人参→扶正 |
| **特点** | 温阳+散寒+除湿止痛 |
| **禁忌** | 口渴口苦、阴虚发热禁用 |

**记住**：附子汤治少阴阳虚寒凝，关键是"背恶寒+身痛+骨节痛"。痛是其特点。
