# 大青龙汤药证详解

> 大青龙汤为太阳表实兼里热主方，发汗解表、清热除烦。

---

## 一、《伤寒论》原文

> "太阳中风，脉浮紧，发热恶寒，身疼痛，不汗出而烦躁者，大青龙汤主之。"（38条）

> "伤寒脉浮缓，身不疼，但重，乍有轻时，无少阴证者，大青龙汤主之。"（39条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 麻黄 | 六两（去节） | 发汗解表 | 肺、膀胱 |
| 桂枝 | 二两（去皮） | 解肌发表 | 心、肺、膀胱 |
| 甘草 | 二两（炙） | 调和诸药 | 心、肺、脾 |
| 杏仁 | 四十枚（去皮尖） | 降气平喘 | 肺 |
| 生姜 | 三两 | 助解表 | 肺、脾、胃 |
| 大枣 | 十枚 | 补中益气 | 脾、胃 |
| 石膏 | 如鸡子大（碎） | 清热除烦 | 肺、胃 |

**煮服法**：上七味，以水九升，先煮麻黄，减二升，去上沫，内诸药，煮取三升，去滓，温服一升。取微似汗。汗出多者，温粉粉之。一服汗者，停后服。若复服，汗多亡阳遂虚，恶风烦躁不得眠也。

---

## 三、方证要点

### 核心病机

太阳表实兼里热——风寒束表，内有郁热。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 发热恶寒 | 发热、怕冷 | 太阳表证 |
| 身疼痛 | 身体疼痛 | 寒凝经脉 |
| 无汗 | 无汗出 | 表实 |
| 烦躁 | 烦躁不安 | 里热扰心 |
| 脉浮紧 | 脉浮紧 | 表实 |
| 或身重 | 身体沉重 | 湿邪（39条） |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 表实+烦躁 | 高 | 方证匹配，可用 |
| 无烦躁 | — | 改用麻黄汤 |
| 有汗 | — | 改用桂枝汤 |

---

## 四、药证校验

### 麻黄——发汗解表（剂量加倍）

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 发汗 | 无汗 | 表实 ✓ |
| 解表 | 发热恶寒 | 风寒束表 ✓ |

**麻黄药证**：凡见无汗、发热恶寒者，麻黄有据。

### 石膏——清热除烦

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 清热 | 烦躁 | 里热扰心 ✓ |
| 除烦 | 烦躁不安 | 热扰心神 ✓ |

**石膏药证**：凡见烦躁者，石膏有据。

### 桂枝——解肌发表

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 解表 | 发热恶寒 | 表邪 ✓ |

### 杏仁——降气平喘

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 降气 | 咳喘（或然） | 肺气上逆 ✓ |

---

## 五、禁忌 ⚠️

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 高血压、心脏病 | 麻黄量加倍，升压明显 | 禁用 |
| 体虚、老人、孕妇 | 发汗太猛，易伤正 | 禁用或慎用 |
| 有汗 | 表虚证 | 改用桂枝汤 |
| 少阴证（但欲寐、脉微） | 误汗致亡阳 | 禁用 |

**⚠️ 大青龙汤发汗力最强，用前必问：高血压、心脏病、体虚！**

---

## 六、典型案例

### 案例：太阳表实兼里热

**患者**：男，25岁

**主诉**：发热恶寒、无汗、烦躁2天

**十问**：
- 吃：不想吃
- 喝：口渴
- 拉：正常
- 撒：尿黄
- 睡：烦躁睡不着
- 汗：无汗
- 痛：身痛
- 寒热：发热恶寒明显

**舌脉**：舌红苔薄黄，脉浮紧

**辨证**：太阳表实兼里热

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 发热恶寒 | 发热恶寒 ✓ | ✓ |
| 身疼痛 | 身痛 ✓ | ✓ |
| 无汗 | 无汗 ✓ | ✓ |
| 烦躁 | 烦躁失眠 ✓ | ✓ |
| 脉浮紧 | 脉浮紧 ✓ | ✓ |

**禁忌排查**：无高血压、无心脏病、体健 ✓

**处方**：大青龙汤
- 麻黄 18g（剂量大）
- 桂枝 6g
- 炙甘草 6g
- 杏仁 9g
- 生姜 9g
- 大枣 4枚
- 石膏 30g

**医嘱**：温服，取微似汗，汗出即止。不可大汗！

## 七、方剂对比

### 大青龙汤 vs 麻黄汤

| 对比项 | 大青龙汤 | 麻黄汤 |
|-------|---------|-------|
| 烦躁 | 有 | 无 |
| 石膏 | 有 | 无 |
| 麻黄量 | 六两（加倍） | 二两 |
| 生姜大枣 | 有 | 无 |
| 发汗力 | 最强 | 强 |

**鉴别要点**：有无烦躁（里热）。

### 大青龙汤 vs 小青龙汤

| 对比项 | 大青龙汤 | 小青龙汤 |
|-------|---------|---------|
| 病机 | 表实+里热 | 表实+寒饮 |
| 核心症状 | 烦躁 | 咳喘+痰饮 |
| 石膏 | 有 | 无 |
| 干姜细辛五味子半夏 | 无 | 有 |

## 八、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 发热恶寒+无汗+身痛+烦躁 |
| **核心病机** | 太阳表实兼里热 |
| **核心药证** | 麻黄→无汗；石膏→烦躁 |
| **特点** | 发汗力最强，兼清里热 |
| **禁忌** | 高血压、心脏病、体虚禁用；有汗禁用 |
| **注意** | 汗出即止，不可过服 |

**记住**：大青龙汤发汗力最强，关键是"无汗+烦躁"。用前必问高血压、心脏病！
