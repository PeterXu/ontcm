# 麻黄汤药证详解

> 麻黄汤为太阳表实证主方，发汗解表、宣肺平喘。

---

## 一、《伤寒论》原文

> "太阳病，头痛发热，身疼腰痛，骨节疼痛，恶风无汗而喘者，麻黄汤主之。"（35条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 麻黄 | 二两（去节） | 发汗解表、宣肺平喘 | 肺、膀胱 |
| 桂枝 | 二两（去皮） | 解肌发表、温通经脉 | 心、肺、膀胱 |
| 杏仁 | 七十个（去皮尖） | 降气平喘 | 肺 |
| 甘草（炙） | 一两 | 调和诸药 | 心、肺、脾 |

**煮服法**：上四味，以水九升，先煮麻黄减二升，去上沫，内诸药，煮取二升半，去滓，温服八合。覆取微似汗，不须啜粥，余如桂枝法将息。

---

## 三、方证要点

### 核心病机

太阳表实证——风寒束表，腠理闭塞，肺气不宣。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 恶寒（恶风） | 怕冷明显 | 寒邪束表 |
| 无汗 | 无汗出 | 腠理闭塞 |
| 发热 | 发热 | 正邪抗争 |
| 头痛 | 头痛 | 寒凝太阳经 |
| 身疼 | 身体疼痛 | 寒凝经脉 |
| 骨节疼痛 | 关节疼痛 | 寒凝关节 |
| 喘 | 呼吸困难、喘促 | 寒束肺气 |
| 脉浮紧 | 脉浮而紧 | 表寒实证 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| ≥3条（含无汗+脉浮紧） | 高 | 方证匹配，可用 |
| 有汗 | — | 改用桂枝汤 |
| 脉浮缓 | — | 改用桂枝汤 |

**关键鉴别点**：无汗 + 脉浮紧 = 表实证

---

## 四、药证校验

### 麻黄——发汗解表、宣肺平喘

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 发汗解表 | 无汗、恶寒 | 寒邪束表 ✓ |
| 宣肺 | 咳喘 | 寒束肺气 ✓ |
| 利水 | 小便不利（或然） | 肺气不宣，水道不通 ✓ |

**麻黄药证**：凡见无汗、恶寒、喘者，麻黄有据。

### 桂枝——解肌发表、温通经脉

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 解表 | 发热、恶寒 | 表邪 ✓ |
| 温经 | 头痛、身痛 | 寒凝经脉 ✓ |
| 通阳 | 无汗 | 配合麻黄开腠理 ✓ |

**桂枝药证**：凡见头痛、身痛、恶寒者，桂枝有据。

### 杏仁——降气平喘

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 降气 | 咳喘 | 肺气上逆 ✓ |
| 平喘 | 呼吸困难 | 寒束肺气 ✓ |

**杏仁药证**：凡见咳喘者，杏仁有据。

### 炙甘草——调和诸药

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 调和 | — | 使麻黄发汗不伤正 ✓ |

## 五、加减法

### 加法

| 加药 | 适用症状 | 加药依据 |
|------|---------|---------|
| 葛根 | 项背强几几 | 葛根解肌舒筋 |
| 白术 | 小便不利 | 白术健脾利水 |
| 附子 | 恶寒甚、脉沉 | 温阳解表 |

### 减法或调整

| 调整 | 适用情况 | 调整依据 |
|------|---------|---------|
| 去杏仁 | 无咳喘 | 无肺气上逆 |
| 麻黄减量 | 汗出过多 | 减发汗力 |
| 改用桂枝汤 | 有汗 | 表虚证 |

### 常用加减方

| 加减方 | 组成 | 适用 |
|-------|------|------|
| 葛根汤 | 麻黄汤+葛根 | 太阳阳明合病、项背强 |
| 麻黄加术汤 | 麻黄汤+白术 | 太阳表实+湿邪 |
| 小青龙汤 | 麻黄+桂枝+干姜+细辛+五味子+半夏 | 太阳表实+寒饮内停 |

---

## 六、禁忌 ⚠️

| 禁忌人群 | 原因 | 替代方案 |
|---------|------|---------|
| 高血压患者 | 麻黄升压，兴奋心脏 | 荆防败毒散等温和解表方 |
| 心脏病患者 | 麻黄兴奋心脏，加重心律问题 | 温和方剂 |
| 体虚者、老人 | 发汗太过伤正气 | 桂枝汤或参苏饮 |
| 孕妇 | 发汗伤胎 | 温和方剂 |
| 失眠、焦虑患者 | 麻黄兴奋中枢 | 温和方剂 |
| 已汗出者 | 表虚证，方证不符 | 桂枝汤 |
| 脉浮缓者 | 表虚证 | 桂枝汤 |

**⚠️ 用麻黄汤前必须询问：高血压、心脏病、失眠史！**

## 七、典型案例

### 案例：太阳表实证

**患者**：男，28岁

**主诉**：恶寒发热1天

**十问**：
- 吃：正常
- 喝：口不渴
- 拉：正常
- 撒：正常
- 睡：正常
- 汗：无汗
- 痛：头痛、身痛明显
- 寒热：恶寒重、发热

**舌脉**：苔薄白，脉浮紧

**辨证**：太阳表实证

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 恶寒 | 恶寒重 ✓ | ✓ |
| 无汗 | 无汗 ✓ | ✓ |
| 发热 | 发热 ✓ | ✓ |
| 头痛身痛 | 头痛身痛 ✓ | ✓ |
| 脉浮紧 | 脉浮紧 ✓ | ✓ |

**禁忌排查**：无高血压、无心脏病、无失眠史 ✓

**处方**：麻黄汤
- 麻黄 9g
- 桂枝 6g
- 杏仁 9g
- 炙甘草 3g

**医嘱**：温服，覆被取微汗，汗出即止，不可大汗。

## 八、方剂对比

### 麻黄汤 vs 桂枝汤

| 对比项 | 麻黄汤 | 桂枝汤 |
|-------|-------|-------|
| 病机 | 太阳表实（腠理闭塞） | 太阳表虚（腠理疏松） |
| 汗出 | 无汗 ✓ | 有汗 ✓ |
| 恶寒程度 | 重 | 轻（恶风） |
| 身痛 | 明显 | 较轻 |
| 脉象 | 浮紧 ✓ | 浮缓 ✓ |
| 麻黄 | 有 | 无 |
| 桂枝量 | 二两 | 三两 |
| 芍药 | 无 | 有 |

**鉴别要点**：有汗无汗 + 脉紧脉缓

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 无汗+恶寒+发热+头痛身痛+脉浮紧 |
| **核心病机** | 太阳表实（风寒束表，腠理闭塞） |
| **核心药证** | 麻黄→无汗、喘；桂枝→头痛身痛；杏仁→咳喘 |
| **禁忌** | 高血压、心脏病、体虚、孕妇、失眠者禁用 |
| **鉴别** | 与桂枝汤鉴别关键：有汗无汗 |

**记住**：麻黄汤是太阳表实主方，用前必问高血压、心脏病！有汗者改用桂枝汤。
