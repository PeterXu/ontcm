# 小青龙汤药证详解

> 小青龙汤为太阳表实兼寒饮主方，解表散寒、温肺化饮。

---

## 一、《伤寒论》原文

> "伤寒表不解，心下有水气，干呕，发热而咳，或渴，或利，或噎，或小便不利、少腹满，或喘者，小青龙汤主之。"（40条）

> "伤寒心下有水气，咳而微喘，发汗不愈者，小青龙汤主之。"（41条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 麻黄 | 三两（去节） | 发汗解表 | 肺、膀胱 |
| 桂枝 | 三两（去皮） | 解肌发表 | 心、肺、膀胱 |
| 干姜 | 三两 | 温肺化饮 | 脾、胃、心、肺 |
| 细辛 | 三两 | 温肺化饮 | 肺、肾 |
| 五味子 | 半升 | 敛肺止咳 | 肺、心、肾 |
| 半夏 | 半升（洗） | 降逆化痰 | 脾、胃 |
| 芍药 | 三两 | 敛阴和营 | 肝、脾 |
| 甘草 | 三两（炙） | 调和诸药 | 心、肺、脾 |

**煮服法**：上八味，以水一斗，先煮麻黄，减二升，去上沫，内诸药，煮取三升，去滓，温服一升。

---

## 三、方证要点

### 核心病机

太阳表实兼寒饮内停——风寒束表，肺失宣降，水饮内停。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 发热恶寒 | 发热、怕冷 | 太阳表证 |
| 无汗 | 无汗出 | 表实 |
| 咳喘 | 咳嗽、气喘 | 寒饮犯肺 |
| 干呕 | 恶心干呕 | 水饮犯胃 |
| 或渴 | 口渴（水饮内停） | 气不化津 |
| 或利 | 腹泻 | 水饮下注 |
| 或小便不利 | 小便少 | 水饮内停 |
| 苔白滑 | 舌苔白滑 | 寒饮内停 |
| 脉浮紧 | 脉浮紧 | 表实 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 表实+咳喘+寒饮征象 | 高 | 方证匹配，可用 |
| 无咳喘 | — | 改用麻黄汤 |
| 有汗 | — | 改用桂枝汤 |

## 四、药证校验

### 麻黄——发汗解表

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 发汗 | 无汗 | 表实 ✓ |
| 宣肺 | 咳喘 | 肺气不宣 ✓ |

### 干姜、细辛——温肺化饮

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温肺 | 咳喘痰白 | 寒饮犯肺 ✓ |
| 化饮 | 苔白滑 | 水饮内停 ✓ |

**干姜细辛药证**：凡见咳喘痰白、苔白滑者，干姜细辛有据。

### 五味子——敛肺止咳

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 敛肺 | 咳喘 | 肺气上逆 ✓ |
| 止咳 | 咳嗽 | 配合干姜细辛 ✓ |

**五味子药证**：凡见咳喘者，五味子有据。

### 半夏——降逆化痰

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 降逆 | 干呕 | 水饮犯胃 ✓ |
| 化痰 | 咳痰 | 水饮 ✓ |

**半夏药证**：凡见干呕、咳痰者，半夏有据。

---

## 五、加减法

| 或然证 | 加减 | 依据 |
|-------|------|------|
| 渴 | 去半夏，加瓜蒌根 | 渴伤津 |
| 利 | 去麻黄，加荛花 | 利下去麻黄免发汗 |
| 噎 | 去麻黄，加附子 | 噎去麻黄 |
| 小便不利、少腹满 | 去麻黄，加茯苓 | 利水 |
| 喘 | 去麻黄，加杏仁 | 降气平喘 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 高血压、心脏病 | 含麻黄 | 改用温和方 |
| 有汗 | 表虚证 | 改用桂枝汤 |
| 纯里热（痰黄、舌红） | 寒饮方 | 改用清热化痰方 |

---

## 七、典型案例

### 案例：太阳表实兼寒饮

**患者**：男，40岁

**主诉**：发热恶寒、咳喘3天

**十问**：
- 吃：不想吃
- 喝：口渴不想喝
- 拉：稀便
- 撒：小便少
- 睡：咳喘影响睡眠
- 汗：无汗
- 痛：无
- 寒热：发热恶寒

**舌脉**：苔白滑，脉浮紧

**辨证**：太阳表实兼寒饮内停

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 发热恶寒 | 发热恶寒 ✓ | ✓ |
| 无汗 | 无汗 ✓ | ✓ |
| 咳喘 | 咳喘 ✓ | ✓ |
| 干呕 | 干呕 ✓ | ✓ |
| 渴不欲饮 | 口渴不想喝 ✓ | ✓ |
| 小便不利 | 小便少 ✓ | ✓ |
| 苔白滑 | 苔白滑 ✓ | ✓ |

**处方**：小青龙汤
- 麻黄 9g
- 桂枝 9g
- 干姜 9g
- 细辛 3g
- 五味子 9g
- 半夏 9g
- 芍药 9g
- 炙甘草 9g

## 八、方剂对比

### 小青龙汤 vs 大青龙汤

| 对比项 | 小青龙汤 | 大青龙汤 |
|-------|---------|---------|
| 病机 | 表实+寒饮 | 表实+里热 |
| 核心症状 | 咳喘+痰饮 | 烦躁 |
| 石膏 | 无 | 有 |
| 干姜细辛五味子半夏 | 有 | 无 |

**鉴别要点**：寒饮（咳喘痰白）vs 里热（烦躁）。

### 小青龙汤 vs 麻黄汤

| 对比项 | 小青龙汤 | 麻黄汤 |
|-------|---------|-------|
| 核心症状 | 咳喘+寒饮 | 身痛+喘 |
| 干姜细辛五味子半夏 | 有 | 无 |

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 发热恶寒+无汗+咳喘+苔白滑 |
| **核心病机** | 太阳表实兼寒饮内停 |
| **核心药证** | 麻黄→无汗；干姜细辛→寒饮；五味子→咳喘；半夏→干呕 |
| **特点** | 解表+温肺化饮 |
| **禁忌** | 高血压、心脏病慎用；有汗禁用；痰黄热痰禁用 |

**记住**：小青龙汤治太阳表实兼寒饮，关键是"无汗+咳喘+苔白滑"。痰黄者禁用。
