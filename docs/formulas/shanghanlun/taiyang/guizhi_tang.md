# 桂枝汤药证详解

> 桂枝汤为太阳表虚证主方，解肌发表、调和营卫。

---

## 一、《伤寒论》原文

> "太阳病，头痛，发热，汗出，恶风，桂枝汤主之。"（13条）

> "太阳病，初服桂枝汤，反烦不解者，先刺风池、风府，却与桂枝汤则愈。"（24条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 桂枝 | 三两（去皮） | 解肌发表、温通经脉 | 心、肺、膀胱 |
| 芍药 | 三两 | 益阴敛营、缓急止痛 | 肝、脾 |
| 甘草（炙） | 二两 | 调和诸药 | 心、肺、脾 |
| 生姜 | 三两 | 助桂枝解表、和胃 | 肺、脾、胃 |
| 大枣 | 十二枚 | 补中益气、助芍药益阴 | 脾、胃 |

**煮服法**：上五味，以水七升，微火煮取三升，去滓，适寒温，服一升。服已须臾，啜热稀粥一升余，以助药力。温覆令一时许，遍身漐漐微似有汗者益佳，不可令如水流漓，病必不除。若一服汗出病差，停后服，不必尽剂。

---

## 三、方证要点

### 核心病机

太阳表虚证——风寒袭表，营卫不和，腠理疏松。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 发热 | 发热 | 正邪抗争 |
| 恶风 | 怕风（比恶寒轻） | 风邪袭表 |
| 汗出 | 有汗出 | 腠理疏松，营卫不和 |
| 头痛 | 头痛 | 风邪上扰 |
| 脉浮缓 | 脉浮而缓 | 表虚证 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| ≥3条（含汗出+脉浮缓） | 高 | 方证匹配，可用 |
| 无汗 | — | 改用麻黄汤 |
| 脉浮紧 | — | 改用麻黄汤 |

**关键鉴别点**：有汗 + 脉浮缓 = 表虚证

---

## 四、药证校验

### 桂枝——解肌发表、温通经脉

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 解肌发表 | 发热、恶风 | 表邪 ✓ |
| 温经 | 头痛 | 风邪上扰 ✓ |
| 通阳 | 汗出（调和营卫） | 腠理疏松 ✓ |

**桂枝药证**：凡见发热、恶风、头痛者，桂枝有据。

### 芍药——益阴敛营

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 敛营 | 汗出 | 营阴外泄 ✓ |
| 缓急 | 腹痛（或然） | 筋脉拘急 ✓ |

**芍药药证**：凡见汗出（营阴外泄）者，芍药有据。

### 生姜——助桂枝解表

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 解表 | 恶风 | 配合桂枝 ✓ |
| 和胃 | 呕（或然） | 胃气不和 ✓ |

**生姜药证**：凡见恶风者，生姜有据。

### 大枣、炙甘草——补中调和

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补中 | 食少（或然） | 调和中焦 ✓ |
| 调和营卫 | 汗出 | 配合桂枝芍药 ✓ |

---

## 五、加减法

### 常用加减

| 加减方 | 组成变化 | 适用 |
|-------|---------|------|
| 桂枝加葛根汤 | +葛根 | 项背强几几 |
| 桂枝加厚朴杏子汤 | +厚朴、杏仁 | 兼咳喘 |
| 桂枝加附子汤 | +附子 | 汗出过多、恶风甚 |
| 桂枝加芍药汤 | 芍药加倍 | 腹痛 |
| 桂枝加大黄汤 | +大黄 | 兼便秘 |
| 桂枝去芍药汤 | 去芍药 | 胸满 |
| 小建中汤 | 芍药加倍+饴糖 | 腹中时痛、心悸 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 无汗（表实证） | 方证不符 | 改用麻黄汤 |
| 脉浮紧（表实证） | 方证不符 | 改用麻黄汤 |
| 阳明里热（口渴喜凉） | 表里同病 | 先表后里，或表里双解 |
| 酗酒者 | 酒后易伤津 | 慎用发汗方 |

---

## 七、典型案例

### 案例：太阳表虚证

**患者**：女，32岁

**主诉**：发热、怕风2天

**十问**：
- 吃：正常
- 喝：口不渴
- 拉：正常
- 撒：正常
- 睡：正常
- 汗：有汗出
- 痛：轻微头痛
- 寒热：怕风，发热不重

**舌脉**：苔薄白，脉浮缓

**辨证**：太阳表虚证

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 发热 | 发热 ✓ | ✓ |
| 恶风 | 怕风 ✓ | ✓ |
| 汗出 | 有汗 ✓ | ✓ |
| 头痛 | 轻微头痛 ✓ | ✓ |
| 脉浮缓 | 脉浮缓 ✓ | ✓ |

**处方**：桂枝汤
- 桂枝 9g
- 芍药 9g
- 炙甘草 6g
- 生姜 9g
- 大枣 4枚

**医嘱**：温服，啜热稀粥，覆被取微汗。

## 八、方剂对比

### 桂枝汤 vs 麻黄汤

| 对比项 | 桂枝汤 | 麻黄汤 |
|-------|-------|-------|
| 病机 | 太阳表虚 | 太阳表实 |
| 汗出 | 有汗 ✓ | 无汗 ✓ |
| 恶寒程度 | 轻（恶风） | 重 |
| 脉象 | 浮缓 ✓ | 浮紧 ✓ |
| 麻黄 | 无 | 有 |
| 芍药 | 有 | 无 |
| 啜粥 | 需啜热粥 | 不须啜粥 |

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 有汗+恶风+发热+头痛+脉浮缓 |
| **核心病机** | 太阳表虚（营卫不和） |
| **核心药证** | 桂枝→发热恶风；芍药→汗出敛营；生姜→助解表 |
| **禁忌** | 无汗者改用麻黄汤 |
| **服法特点** | 需啜热稀粥助药力 |

**记住**：桂枝汤是太阳表虚主方，关键鉴别点是有汗+脉浮缓。服后需啜热稀粥。
