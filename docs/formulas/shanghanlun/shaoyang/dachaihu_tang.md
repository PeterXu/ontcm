# 大柴胡汤药证详解

> 大柴胡汤为少阳阳明合病主方，和解少阳、通下阳明。

---

## 一、《伤寒论》原文

> "太阳病，过经十余日，反二三下之，后四五日，柴胡证仍在者，先与小柴胡汤。呕不止，心下急，郁郁微烦者，为未解也，与大柴胡汤下之则愈。"（103条）

> "伤寒发热，汗出不解，心中痞硬，呕吐而下利者，大柴胡汤主之。"（165条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 柴胡 | 半斤 | 疏解少阳 | 肝、胆 |
| 黄芩 | 三两 | 清少阳热 | 肝、胆、肺 |
| 芍药 | 三两 | 缓急止痛 | 肝、脾 |
| 半夏 | 半升（洗） | 降逆止呕 | 脾、胃 |
| 生姜 | 五两 | 和胃止呕 | 肺、脾、胃 |
| 枳实 | 四枚（炙） | 破气消积 | 脾、胃 |
| 大黄 | 二两 | 攻下通便 | 脾、胃、大肠 |
| 大枣 | 十二枚 | 补中益气 | 脾、胃 |

**煮服法**：上八味，以水一斗二升，煮取六升，去滓，再煎取三升，温服一升，日三服。

**注意**：大柴胡汤去人参、甘草（小柴胡汤原有），加枳实、大黄。

---

## 三、方证要点

### 核心病机

少阳阳明合病——少阳枢机不利，阳明腑气不通。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 往来寒热 | 忽冷忽热 | 少阳证 |
| 胸胁苦满 | 胸胁胀满 | 少阳经气滞 |
| 呕不止 | 呕吐不止 | 胆热犯胃 |
| 心下急 | 心下急痛 | 阳明腑实 |
| 郁郁微烦 | 烦躁 | 热扰心神 |
| 便秘或下利 | 大便不通或热利 | 阳明腑实 |
| 苔黄 | 苔黄 | 里热 |
| 脉弦数 | 脉弦数 | 少阳+阳明热 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 少阳证+阳明腑实 | 高 | 方证匹配，可用 |
| 无阳明腑实 | — | 改用小柴胡汤 |

---

## 四、药证校验

### 柴胡——疏解少阳

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 疏解少阳 | 往来寒热 | 少阳证 ✓ |
| 疏肝 | 胸胁苦满 | 少阳经气滞 ✓ |

### 枳实、大黄——攻下通腑

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 破气 | 心下急 | 腑气不通 ✓ |
| 攻下 | 便秘或下利热臭 | 阳明腑实 ✓ |

**枳实大黄药证**：凡见心下急、便秘者，枳实大黄有据。

### 半夏、生姜——降逆止呕

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 止呕 | 呕不止 | 胆热犯胃 ✓ |

### 芍药——缓急止痛

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 缓急 | 心下急痛 | 筋脉拘急 ✓ |

---

## 五、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 纯少阳证（无腑实） | 无需攻下 | 改用小柴胡汤 |
| 体虚者 | 攻下伤正 | 改用小柴胡汤或加扶正药 |
| 孕妇 | 攻下伤胎 | 慎用 |

---

## 六、典型案例

### 案例：少阳阳明合病

**患者**：男，45岁

**主诉**：发热、腹痛、便秘5天

**十问**：
- 吃：不想吃
- 喝：口苦
- 拉：便秘，5天未大便
- 撒：尿黄
- 睡：心烦
- 汗：有时出汗
- 痛：胸胁胀满、心下急痛
- 寒热：往来寒热

**舌脉**：苔黄，脉弦数

**辨证**：少阳阳明合病

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 往来寒热 | 往来寒热 ✓ | ✓ |
| 胸胁苦满 | 胸胁胀满 ✓ | ✓ |
| 呕不止 | 恶心 ✓ | ✓ |
| 心下急 | 心下急痛 ✓ | ✓ |
| 郁郁微烦 | 心烦 ✓ | ✓ |
| 便秘 | 便秘 ✓ | ✓ |
| 苔黄脉弦数 | 苔黄脉弦数 ✓ | ✓ |

**处方**：大柴胡汤
- 柴胡 15g
- 黄芩 9g
- 芍药 9g
- 半夏 9g
- 生姜 15g
- 枳实 9g
- 大黄 6g（后下）
- 大枣 4枚

## 七、方剂对比

### 大柴胡汤 vs 小柴胡汤

| 对比项 | 大柴胡汤 | 小柴胡汤 |
|-------|---------|---------|
| 病机 | 少阳+阳明腑实 | 纯少阳证 |
| 心下急 | 有 | 无 |
| 便秘 | 常有 | 无 |
| 人参甘草 | 无 | 有 |
| 枳实大黄 | 有 | 无 |

**鉴别要点**：有无腑实（心下急、便秘）。

### 大柴胡汤 vs 大承气汤

| 对比项 | 大柴胡汤 | 大承气汤 |
|-------|---------|---------|
| 少阳证 | 有 | 无 |
| 往来寒热胸胁苦满 | 有 | 无 |
| 柴胡黄芩 | 有 | 无 |

**鉴别要点**：有无少阳证。

---

## 八、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 小柴胡汤证+心下急+便秘（少阳证+阳明腑实） |
| **核心病机** | 少阳阳明合病 |
| **核心药证** | 柴胡黄芩→少阳；枳实大黄→腑实 |
| **特点** | 和解少阳+通下阳明 |
| **禁忌** | 纯少阳证改小柴胡汤；体虚者慎用 |

**记住**：大柴胡汤治少阳阳明合病，关键是"小柴胡汤证+腑实征象（心下急、便秘）"。
