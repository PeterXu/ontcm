# 小柴胡汤药证详解

> 小柴胡汤为少阳病主方，和解少阳、疏利肝胆。

---

## 一、《伤寒论》原文

> "伤寒五六日中风，往来寒热，胸胁苦满，嘿嘿不欲饮食，心烦喜呕，或胸中烦而不呕，或渴，或腹中痛，或胁下痞鞕，或心下悸、小便不利，或不渴、身有微热，或咳者，小柴胡汤主之。"（96条）

> "但见一证便是，不必悉具。"（101条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 柴胡 | 半斤 | 疏解少阳、退热 | 肝、胆 |
| 黄芩 | 三两 | 清少阳热 | 肝、胆、肺 |
| 人参 | 三两 | 补气扶正 | 脾、肺 |
| 半夏 | 半升 | 降逆止呕 | 脾、胃 |
| 甘草（炙） | 三两 | 调和诸药 | 心、肺、脾 |
| 生姜 | 三两 | 和胃止呕 | 肺、脾、胃 |
| 大枣 | 十二枚 | 补中益气 | 脾、胃 |

**煮服法**：上七味，以水一斗二升，煮取六升，去滓，再煎取三升，温服一升，日三服。

## 三、方证要点

### 核心病机

少阳枢机不利——邪居半表半里，正邪分争，枢机不利，胆火上炎。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 往来寒热 | 忽冷忽热 | 邪居半表半里，正邪分争 |
| 胸胁苦满 | 胸胁胀闷不适 | 少阳经循行部位气滞 |
| 嘿嘿不欲饮食 | 精神不振、不想吃 | 胆火内郁，胃气不和 |
| 心烦喜呕 | 心烦、恶心呕吐 | 胆热犯胃 |
| 口苦 | 口苦 | 胆火上炎 |
| 咽干 | 咽干 | 胆火伤津 |
| 目眩 | 目眩头晕 | 胆火上扰清窍 |
| 舌苔薄黄 | 苔薄黄 | 少阳有热 |
| 脉弦 | 脉弦 | 少阳气滞 |

### 方证匹配原则

**"但见一证便是，不必悉具"**

核心症状（见到其中之一即可考虑小柴胡汤）：
1. 往来寒热 ✓
2. 胸胁苦满 ✓
3. 嘿嘿不欲饮食 ✓
4. 心烦喜呕 ✓
5. 口苦 ✓

## 四、药证校验

### 柴胡——疏解少阳

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 疏解少阳 | 往来寒热 | 半表半里热 ✓ |
| 疏肝 | 胸胁苦满 | 少阳经气滞 ✓ |
| 退热 | 发热 | 少阳热 ✓ |

**柴胡药证**：凡见往来寒热、胸胁苦满者，柴胡有据。

### 黄芩——清少阳热

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 清热 | 口苦、咽干 | 胆火上炎 ✓ |
| 清肺 | 咳嗽（或然证） | 肺热 ✓ |
| 清肝 | 心烦 | 胆热扰心 ✓ |

**黄芩药证**：凡见口苦、心烦、苔黄者，黄芩有据。

### 人参——补气扶正

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补气 | 乏力、正气虚 | 扶正祛邪 ✓ |
| 健脾 | 不欲饮食 | 脾气虚 ✓ |

**人参药证**：凡见乏力、食少、正气不足者，人参有据。（少阳病正邪分争，需扶正）

### 半夏——降逆止呕

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 止呕 | 喜呕、恶心 | 胆热犯胃 ✓ |
| 化痰 | 咳嗽有痰（或然证） | 痰饮 ✓ |

**半夏药证**：凡见恶心呕吐者，半夏有据。

### 生姜——和胃止呕

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 止呕 | 喜呕 | 配合半夏 ✓ |
| 解表 | 微热（或然证） | 表邪未尽 ✓ |

**生姜药证**：凡见恶心呕吐者，生姜有据。

### 大枣、炙甘草——补中调和

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补中 | 食少 | 辅助人参 ✓ |
| 调和诸药 | — | 使药力缓和 ✓ |

## 五、加减法

### 或然证加减（《伤寒论》原方加减）

| 或然证 | 加减 | 依据 |
|-------|------|------|
| 胸中烦而不呕 | 去半夏、人参，加瓜蒌 | 无呕去半夏；烦加瓜蒌清热 |
| 渴 | 去半夏，加人参、天花粉 | 渴伤津，加天花粉生津 |
| 腹中痛 | 去黄芩，加芍药 | 腹痛芍药缓急；黄芩苦寒伤脾 |
| 胁下痞硬 | 去大枣，加牡蛎 | 牡蛎软坚散结 |
| 心下悸、小便不利 | 去黄芩，加茯苓 | 茯苓利水宁心 |
| 不渴、身有微热 | 去人参，加桂枝 | 表邪未尽，桂枝解表 |
| 咳 | 去人参、大枣、生姜，加五味子、干姜 | 五味子敛肺，干姜温肺 |

### 常用加减方

| 加减方 | 组成 | 适用 |
|-------|------|------|
| 大柴胡汤 | 小柴胡汤去人参、甘草，加枳实、芍药、大黄 | 少阳+阳明腑实 |
| 柴胡桂枝汤 | 小柴胡汤+桂枝汤 | 少阳+太阳表证 |
| 柴胡加芒硝汤 | 小柴胡汤+芒硝 | 少阳+阳明腑实（较轻） |
| 柴胡加龙骨牡蛎汤 | 小柴胡汤+龙骨、牡蛎、桂枝、茯苓、大黄、铅丹 | 少阳+惊悸、谵语 |

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 纯太阳表证 | 方证不符（半表半里方） | 辨证为太阳，用麻黄/桂枝汤 |
| 纯阳明里热 | 方证不符 | 辨证为阳明，用白虎/承气汤 |
| 纯太阴里寒 | 方证不符 | 辨证为太阴，用理中汤 |
| 纯少阴里虚 | 方证不符 | 辨证为少阴，用四逆汤 |

## 七、典型案例

### 案例：少阳病

**患者**：男，35岁

**主诉**：发热、口苦3天

**十问**：
- 吃：不想吃
- 喝：口苦、咽干
- 拉：正常
- 撒：正常
- 睡：心烦睡不着
- 汗：有时出汗
- 痛：胸胁胀满不适
- 寒热：忽冷忽热，上午冷下午热

**舌脉**：苔薄黄，脉弦

**辨证**：少阳病

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 往来寒热 | 忽冷忽热 ✓ | ✓ |
| 胸胁苦满 | 胸胁胀满 ✓ | ✓ |
| 嘿嘿不欲饮食 | 不想吃 ✓ | ✓ |
| 心烦喜呕 | 心烦 ✓ | ✓ |
| 口苦咽干 | 口苦咽干 ✓ | ✓ |

**药证校验**：
| 药味 | 对应症状 | 有据 |
|------|---------|------|
| 柴胡 | 往来寒热、胸胁苦满 | ✓ |
| 黄芩 | 口苦、心烦 | ✓ |
| 人参 | 不想吃、正气需扶 | ✓ |
| 半夏 | 心烦（胆热犯胃） | ✓ |
| 生姜 | 配合半夏 | ✓ |
| 大枣甘草 | 调和补中 | ✓ |

**处方**：小柴胡汤
- 柴胡 24g
- 黄芩 9g
- 人参 9g
- 半夏 12g
- 炙甘草 9g
- 生姜 9g
- 大枣 4枚

**复诊**：发热消退，口苦消失，食欲恢复。

## 八、方剂对比

### 小柴胡汤 vs 大柴胡汤

| 对比项 | 小柴胡汤 | 大柴胡汤 |
|-------|---------|---------|
| 病机 | 少阳枢机不利 | 少阳+阳明腑实 |
| 核心症状 | 往来寒热、胸胁苦满 | 上+腹满痛、便秘 |
| 舌象 | 苔薄黄 | 苔黄厚 |
| 脉象 | 弦 | 弦数 |
| 有无腑实 | 无 | 有 |
| 人参甘草 | 有（扶正） | 无（去扶正） |
| 枳实大黄芍药 | 无 | 有（攻下） |

**鉴别要点**：有无腑实（便秘、腹满痛）。

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 往来寒热+胸胁苦满+口苦+心烦喜呕（但见一证便是） |
| **核心病机** | 少阳枢机不利 |
| **核心药证** | 柴胡→往来寒热、胸胁苦满；黄芩→口苦、心烦；人参→扶正；半夏→呕 |
| **加减原则** | 或然证按原文加减；合病用加减方 |
| **禁忌** | 纯表证、纯里证不用 |

**记住**：小柴胡汤是少阳病主方，"但见一证便是"，不必悉具。关键在于识别半表半里证。
