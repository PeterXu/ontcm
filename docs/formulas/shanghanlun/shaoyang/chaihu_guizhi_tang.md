# 柴胡桂枝汤药证详解

> 柴胡桂枝汤为太阳少阳合病主方，和解少阳、解肌发表。

---

## 一、《伤寒论》原文

> "伤寒六七日，发热微恶寒，支节烦疼，微呕，心下支结，外证未去者，柴胡桂枝汤主之。"（146条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 |
|------|------------|------|
| 柴胡 | 四两 | 疏解少阳 |
| 黄芩 | 一两半 | 清少阳热 |
| 人参 | 一两半 | 补气扶正 |
| 半夏 | 二合半 | 降逆止呕 |
| 甘草 | 二两（炙） | 调和诸药 |
| 桂枝 | 一两半（去皮） | 解肌发表 |
| 芍药 | 一两半 | 敛阴和营 |
| 生姜 | 一两半 | 助解表 |
| 大枣 | 六枚 | 补中益气 |

**煮服法**：上九味，以水七升，煮取三升，去滓，温服一升。

---

## 三、方证要点

### 核心病机

太阳少阳合病——表邪未解，少阳枢机不利。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 发热微恶寒 | 发热、轻微怕冷 | 太阳表证 |
| 支节烦疼 | 关节疼痛 | 表邪 |
| 微呕 | 轻微恶心 | 少阳证 |
| 心下支结 | 心下胀满不适 | 少阳证 |

---

## 四、方剂对比

### 柴胡桂枝汤 vs 小柴胡汤

| 对比项 | 柴胡桂枝汤 | 小柴胡汤 |
|-------|-----------|---------|
| 表证 | 有 | 无 |
| 桂枝芍药 | 有 | 无 |

**鉴别要点**：有无表证。

---

## 五、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 发热微恶寒+支节烦疼+微呕+心下支结 |
| **核心病机** | 太阳少阳合病 |
| **核心药证** | 小柴胡汤+桂枝汤 |
| **特点** | 和解少阳+解表 |

**记住**：柴胡桂枝汤治太阳少阳合病，关键是"表证+少阳证"。
