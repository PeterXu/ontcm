# 乌梅丸药证详解

> 乌梅丸为厥阴病主方，寒热并用、安蛔止痛。

---

## 一、《伤寒论》原文

> "伤寒脉微而厥，至七八日肤冷，其人躁无暂安时者，此为脏厥，非蛔厥也。蛔厥者，其人当吐蛔。今病者静，而复时烦者，此为脏寒。蛔上入其膈，故烦，须臾复止；得食而呕，又烦者，蛔闻食臭出，其人常自吐蛔。蛔厥者，乌梅丸主之。"（338条）

> "又主久利。"（338条）

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 | 寒热属性 |
|------|------------|------|------|---------|
| 乌梅 | 三百枚 | 安蛔止痛、涩肠止泻 | 肝、脾、肺 | 平 |
| 细辛 | 六两 | 温散寒邪 | 肺、肾 | 温 |
| 干姜 | 十两 | 温中散寒 | 脾、胃、心、肺 | 热 |
| 黄连 | 十六两 | 清热燥湿 | 心、脾、胃、肝 | 寒 |
| 当归 | 四两 | 养血活血 | 心、肝、脾 | 温 |
| 附子 | 六两（炮去皮） | 温阳散寒 | 心、肾、脾 | 热 |
| 蜀椒 | 四两（出汗） | 温中杀虫 | 脾、胃 | 温 |
| 桂枝 | 六两（去皮） | 温通经脉 | 心、肺、膀胱 | 温 |
| 人参 | 六两 | 补气健脾 | 脾、肺 | 平 |
| 黄柏 | 六两 | 清热燥湿 | 肾、膀胱 | 寒 |

**制法**：上十味，异捣筛，合治之，以苦酒渍乌梅一宿，去核，蒸之五斗米下，饭熟捣成泥，和药令相得，内臼中，与蜜杵二千下，丸如梧桐子大。

**服法**：先食饮服十丸，日三服，稍加至二十丸。禁生冷、滑物、臭食等。

## 三、方证要点

### 核心病机

厥阴寒热错杂——上热下寒，蛔虫扰动，阴阳气不相顺接。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 消渴 | 口渴多饮 | 上热灼津 |
| 气上撞心 | 胃脘部有气上冲感 | 肝气上逆 |
| 心中疼热 | 胃脘灼热疼痛 | 上热 |
| 饥而不欲食 | 饥饿但不想吃 | 胃热脾寒 |
| 食则吐蛔 | 进食后呕吐蛔虫 | 蛔虫上窜 |
| 时静时烦 | 时而安静时而烦躁 | 蛔虫时动时静 |
| 厥热往来 | 手足厥冷与发热交替 | 阴阳气不相顺接 |
| 久利 | 长期腹泻 | 寒热错杂于肠 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| ≥3条（含寒热错杂特点） | 高 | 方证匹配，可用 |
| 纯寒或纯热 | — | 方证不符，重新辨证 |

## 四、药证校验

### 乌梅——安蛔止痛、涩肠止泻

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 安蛔 | 吐蛔、腹痛 | 蛔虫扰动 ✓ |
| 止痛 | 腹痛 | 蛔虫致痛 ✓ |
| 涩肠 | 久利 | 肠滑不固 ✓ |

**乌梅药证**：凡见吐蛔、腹痛、久利者，乌梅有据。

### 黄连、黄柏——清热燥湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 清热 | 心中疼热、消渴 | 上热 ✓ |
| 燥湿 | 下利 | 湿热下注 ✓ |

**黄连黄柏药证**：凡见心中疼热、消渴、下利热臭者，黄连黄柏有据。

### 干姜、附子、桂枝、细辛、蜀椒——温阳散寒

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温阳 | 四肢厥冷 | 阳虚寒盛 ✓ |
| 散寒 | 腹痛喜温 | 寒凝 ✓ |

**温阳药药证**：凡见厥冷、腹痛喜温者，温阳药有据。

### 人参——补气健脾

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补气 | 乏力、正气虚 | 久病体虚 ✓ |

### 当归——养血活血

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 养血 | 血虚 | 久病伤血 ✓ |

## 五、加减法

| 加减 | 适用症状 | 加减依据 |
|------|---------|---------|
| 重用乌梅 | 吐蛔明显 | 加强安蛔 |
| 重用黄连黄柏 | 热象明显 | 加强清热 |
| 重用附子干姜 | 寒象明显 | 加强温阳 |
| 加使君子、苦楝皮 | 蛔虫明显 | 加强杀虫 |

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 纯热证 | 寒热并用，热证不宜 | 改用清热方 |
| 纯寒证 | 寒热并用，寒证不宜 | 改用温阳方 |
| 孕妇 | 蜀椒、附子有刺激性 | 慎用或改方 |

## 七、典型案例

### 案例：厥阴蛔厥

**患者**：女，45岁

**主诉**：腹痛时作3年，伴吐蛔

**十问**：
- 吃：饥饿但不想吃，食后恶心
- 喝：口渴喜饮
- 拉：时泻时止
- 撒：正常
- 睡：时烦时静
- 汗：正常
- 痛：腹痛时作，有气上冲感
- 寒热：手足时冷时热

**舌脉**：舌红苔白，脉弦

**辨证**：厥阴寒热错杂、蛔厥

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 消渴 | 口渴喜饮 ✓ | ✓ |
| 气上撞心 | 气上冲感 ✓ | ✓ |
| 心中疼热 | 腹痛灼热 ✓ | ✓ |
| 饥不欲食 | 饥不欲食 ✓ | ✓ |
| 吐蛔 | 吐蛔 ✓ | ✓ |
| 时静时烦 | 时烦时静 ✓ | ✓ |

**处方**：乌梅丸
- 乌梅 30g
- 细辛 3g
- 干姜 6g
- 黄连 6g
- 当归 6g
- 附子 6g（先煎）
- 蜀椒 3g
- 桂枝 6g
- 人参 6g
- 黄柏 6g

**医嘱**：丸剂缓服，忌生冷油腻。

## 八、方剂对比

### 乌梅丸 vs 四逆汤

| 对比项 | 乌梅丸 | 四逆汤 |
|-------|-------|-------|
| 病机 | 寒热错杂 | 纯寒证 |
| 核心症状 | 厥热往来+吐蛔 | 但欲寐+肢冷 |
| 寒热属性 | 寒热并用 | 纯热药 |
| 主药 | 乌梅+黄连+附子 | 附子+干姜 |

**鉴别要点**：有无寒热错杂（厥热往来、心中疼热）。

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 消渴+气上撞心+心中疼热+饥不欲食+厥热往来+吐蛔/久利 |
| **核心病机** | 厥阴寒热错杂 |
| **核心药证** | 乌梅→吐蛔腹痛；黄连黄柏→上热；附子干姜→下寒 |
| **特点** | 寒热并用、安蛔止痛 |
| **禁忌** | 纯寒纯热者禁用 |

**记住**：乌梅丸是厥阴寒热错杂主方，关键识别"寒热错杂"特征——上热下寒、厥热往来。
