# 小建中汤药证详解

> 小建中汤为太阴阳阳两虚主方，温中补虚、缓急止痛。

---

## 一、《伤寒论》原文

> "伤寒二三日，心中悸而烦者，小建中汤主之。"（102条）

> "伤寒阳脉涩，阴脉弦，法当腹中急痛，先与小建中汤。"（100条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 桂枝 | 三两（去皮） | 温通经脉 | 心、肺、膀胱 |
| 芍药 | 六两 | 缓急止痛 | 肝、脾 |
| 甘草 | 二两（炙） | 调和补中 | 心、肺、脾 |
| 生姜 | 三两 | 和胃 | 肺、脾、胃 |
| 大枣 | 十二枚 | 补中益气 | 脾、胃 |
| 饴糖 | 一升 | 补中缓急 | 脾、胃 |

**煮服法**：上六味，以水七升，煮取三升，去滓，内饴糖，更上微火消解。温服一升，日三服。

---

## 三、方证要点

### 核心病机

太阴阴阳两虚——中焦虚寒，气血不足，筋脉失养。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 腹中急痛 | 腹痛阵作、喜温按 | 中虚筋脉拘急 |
| 心中悸而烦 | 心悸、心烦 | 气血不足，心神失养 |
| 脉涩弦 | 脉细涩或弦 | 气血不足+拘急 |
| 喜温按 | 腹痛喜温喜按 | 虚寒 |
| 食少 | 食欲差 | 脾虚 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 腹痛+虚象 | 高 | 方证匹配，可用 |
| 腹痛拒按 | — | 实证，改用他方 |

---

## 四、药证校验

### 饴糖——补中缓急

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补中 | 乏力、食少 | 中虚 ✓ |
| 缓急 | 腹中急痛 | 筋脉拘急 ✓ |

**饴糖药证**：凡见腹中急痛者，饴糖有据。

### 芍药（剂量加倍）——缓急止痛

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 缓急 | 腹中急痛 | 筋脉拘急 ✓ |
| 止痛 | 腹痛 | 拘急致痛 ✓ |

**芍药药证**：凡见腹痛拘急者，芍药有据。

### 桂枝——温通经脉

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温通 | 腹痛喜温 | 中焦虚寒 ✓ |
| 通阳 | 心悸 | 心阳不足 ✓ |

### 生姜大枣炙甘草——补中和胃

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补中 | 食少、乏力 | 脾胃虚弱 ✓ |
| 和胃 | 恶心（或然） | 胃气不和 ✓ |

---

## 五、加减法

| 加减方 | 组成变化 | 适用 |
|-------|---------|------|
| 大建中汤 | 去芍药，加蜀椒、干姜、人参 | 腹痛更甚、呕不能食 |
| 黄土汤 | 小建中汤去饴糖+黄土 | 便血 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 腹痛拒按 | 实证 | 改用实证方 |
| 湿热腹痛 | 虚寒方 | 改用清热方 |
| 糖尿病 | 含饴糖 | 改用他方或去饴糖 |

---

## 七、典型案例

### 案例：太阴阴阳两虚

**患者**：女，35岁

**主诉**：腹痛阵作1月

**十问**：
- 吃：食欲差
- 喝：不渴
- 拉：正常
- 撒：正常
- 睡：心悸心烦
- 汗：正常
- 痛：腹中急痛，喜温按
- 寒热：怕冷

**舌脉**：舌淡苔薄白，脉涩弦

**辨证**：太阴阴阳两虚

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 腹中急痛 | 腹痛阵作 ✓ | ✓ |
| 喜温按 | 喜温按 ✓ | ✓ |
| 心中悸而烦 | 心悸心烦 ✓ | ✓ |
| 脉涩弦 | 脉涩弦 ✓ | ✓ |

**处方**：小建中汤
- 桂枝 9g
- 芍药 18g（加倍）
- 炙甘草 6g
- 生姜 9g
- 大枣 4枚
- 饴糖 30g（后下）

## 八、方剂对比

### 小建中汤 vs 理中汤

| 对比项 | 小建中汤 | 理中汤 |
|-------|---------|-------|
| 病机 | 阴阳两虚 | 脾虚寒湿 |
| 核心症状 | 腹中急痛 | 腹满吐利 |
| 饴糖芍药 | 有 | 无 |
| 干姜人参白术 | 无 | 有 |
| 苔象 | 苔薄 | 苔白腻 |

**鉴别要点**：腹痛急痛（拘急）vs 腹满吐利（湿盛）。

### 小建中汤 vs 大建中汤

| 对比项 | 小建中汤 | 大建中汤 |
|-------|---------|---------|
| 腹痛程度 | 较轻 | 更甚 |
| 呕吐 | 无 | 有 |
| 蜀椒干姜人参 | 无 | 有 |

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 腹中急痛+喜温按+心中悸烦+脉涩弦 |
| **核心病机** | 太阴阴阳两虚（中虚筋脉拘急） |
| **核心药证** | 饴糖→补中缓急；芍药（倍量）→腹痛拘急；桂枝→温通 |
| **特点** | 温中补虚+缓急止痛 |
| **禁忌** | 实证腹痛、湿热腹痛、糖尿病 |

**记住**：小建中汤治太阴阴阳两虚，关键是"腹中急痛+喜温按"。芍药倍量是关键。
