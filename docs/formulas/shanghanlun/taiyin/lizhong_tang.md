# 理中汤药证详解

> 理中汤为太阴病主方，温中祛寒、补气健脾。

---

## 一、《伤寒论》原文

> "霍乱，头痛发热，身疼痛，热多欲饮水者，五苓散主之；寒多不用水者，理中汤主之。"（386条）

> "胸痹心中痞，留气结在胸，胸满，胁下逆抢心，枳实薤白桂枝汤主之；人参汤亦主之。"（人参汤即理中汤）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 人参 | 二两 | 补气健脾 | 脾、肺 |
| 白术 | 二两 | 健脾燥湿 | 脾、胃 |
| 干姜 | 二两 | 温中散寒 | 脾、胃、心、肺 |
| 甘草（炙） | 二两 | 补中益气、调和诸药 | 心、肺、脾、胃 |

**煮服法**：上四味，以水八升，煮取三升，去滓，温服一升，日三服。

---

## 三、方证要点

### 核心病机

太阴脾虚寒湿——中焦虚寒，脾失健运，寒湿内停。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 腹满 | 腹胀、腹痛 | 寒凝中焦，气机不畅 |
| 食不下 | 食欲差、不想吃 | 脾虚不运 |
| 自利 | 大便稀、腹泻 | 脾虚湿盛，运化失常 |
| 不渴 | 口不渴、口淡 | 寒证，津液未伤 |
| 喜温 | 喜热饮、腹痛喜温按 | 里寒 |
| 舌淡苔白腻 | 舌质淡、苔白腻 | 寒湿内停 |
| 脉沉弱 | 脉沉而无力 | 里虚寒 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| ≥5条 | 高 | 方证匹配，可用 |
| 3-4条 | 中 | 继续观察，注意合病 |
| <3条 | 低 | 可能方证不符，重新辨证 |

---

## 四、药证校验

### 人参——补气健脾

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补气 | 乏力、少气懒言 | 有气虚表现 ✓ |
| 健脾 | 食欲差、消瘦 | 有脾虚表现 ✓ |
| 生津 | 口干（气虚津不上承） | 气虚津不上承所致口干 ✓ |

**人参药证**：凡见乏力、少气、脉弱、食少者，人参有据。

### 白术——健脾燥湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 健脾 | 食欲差、腹胀 | 有脾虚表现 ✓ |
| 燥湿 | 大便稀、苔白腻 | 有湿象 ✓ |
| 利水 | 小便不利（湿阻） | 有水湿内停 ✓ |

**白术药证**：凡见便稀、苔腻、腹胀、小便不利者，白术有据。

### 干姜——温中散寒

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温中 | 腹痛喜温按 | 有里寒表现 ✓ |
| 散寒 | 怕冷、四肢凉 | 有寒象 ✓ |
| 止呕 | 呕吐清水 | 寒呕 ✓ |
| 止泻 | 腹泻清水样 | 寒泻 ✓ |

**干姜药证**：凡见口淡、喜热饮、腹痛喜温、四肢凉、呕吐清水、腹泻清水者，干姜有据。

### 炙甘草——补中调和

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 补中 | 食少、乏力 | 辅助人参补气 ✓ |
| 调和诸药 | — | 使药力缓和持久 ✓ |

**炙甘草药证**：辅助用药，调和补中，不须特定症状支撑。

---

## 五、加减法

### 加法——症状多一味药的功效

| 加药 | 适用症状 | 加药依据 |
|------|---------|---------|
| 半夏 | 呕吐明显 | 半夏降逆止呕 |
| 附子 | 四肢凉甚、脉微 | 附子温阳救逆 |
| 茯苓 | 小便不利、苔腻甚 | 茯苓利水渗湿 |
| 桂枝 | 兼表证（怕风汗出） | 桂枝解表 |
| 丁香 | 呃逆 | 丁香温中降逆 |

### 减法或调整——症状少一味药的功效

| 调整 | 适用情况 | 调整依据 |
|------|---------|---------|
| 白术炒用 | 便不甚稀 | 减燥性 |
| 白术减量 | 便干或正常 | 无湿象，不需燥湿 |
| 干姜减量 | 轻微寒象 | 寒不甚 |
| 人参改党参 | 轻度气虚 | 气虚不甚，党参代人参 |

### 常用加减方

| 加减方 | 组成 | 适用 |
|-------|------|------|
| 附子理中汤 | 理中汤+附子 | 太阴寒甚、阳虚明显 |
| 理中化痰丸 | 理中汤+半夏、茯苓 | 太阴虚寒+痰湿 |
| 桂枝人参汤 | 理中汤+桂枝 | 太阴+太阳表证 |

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 热证腹泻（大便臭秽、肛门灼热） | 热证用热药加重 | 改用清热止泻方 |
| 阴虚发热 | 温药伤阴 | 改用养阴清热方 |
| 阳明腑实（便秘腹满痛） | 方证不符 | 改用承气汤类 |
| 真热假寒（外寒内热） | 误诊致加重 | 先辨寒热真假 |

## 七、典型案例

### 案例1：太阴脾虚寒（腹泻）

**患者**：女，42岁

**主诉**：腹泻半个月

**十问**：
- 吃：不想吃，吃几口就胀，口淡
- 喝：不渴，想喝热水
- 拉：大便稀，日2-3次
- 撒：小便正常
- 睡：多梦，醒来累
- 汗：无
- 痛：胃胀
- 寒热：怕冷

**舌脉**：舌淡苔白腻，脉沉弱

**辨证**：太阴脾虚寒湿

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 腹满 | 胃胀 ✓ | ✓ |
| 食不下 | 不想吃 ✓ | ✓ |
| 自利 | 大便稀 ✓ | ✓ |
| 不渴 | 不渴 ✓ | ✓ |
| 舌淡苔白腻 | 舌淡苔白腻 ✓ | ✓ |

**药证校验**：
| 药味 | 对应症状 | 有据 |
|------|---------|------|
| 人参 | 乏力、醒来累 | ✓ |
| 白术 | 便稀、苔腻 | ✓ |
| 干姜 | 口淡、喜热、怕冷 | ✓ |
| 炙甘草 | 调和 | ✓ |

**处方**：理中汤
- 人参 9g
- 白术 12g（炒）
- 干姜 9g
- 炙甘草 6g

**复诊**：胃胀减轻，食欲恢复。

## 八、方剂对比

### 理中汤 vs 四逆汤

| 对比项 | 理中汤 | 四逆汤 |
|-------|-------|-------|
| 病位 | 中焦（脾） | 下焦（肾） |
| 病机 | 脾虚寒湿 | 肾阳虚衰 |
| 核心症状 | 腹满吐利 | 但欲寐、肢冷脉微 |
| 脉象 | 沉弱 | 微细或微欲绝 |
| 病势 | 较缓 | 较急，危候 |
| 主药 | 人参、白术、干姜 | 附子、干姜、甘草 |

**鉴别要点**：精神状态、脉象强弱、病势缓急。

### 理中汤 vs 小建中汤

| 对比项 | 理中汤 | 小建中汤 |
|-------|-------|-------|
| 病机 | 脾虚寒湿 | 脾虚阴阳两虚 |
| 核心症状 | 腹满吐利 | 腹中时痛、心悸 |
| 舌象 | 舌淡苔白腻 | 舌淡苔薄 |
| 主药 | 人参、白术、干姜 | 桂枝、芍药、饴糖 |

**鉴别要点**：有无湿象（苔腻、便稀）、腹痛性质。

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 腹满+食不下+自利+不渴+舌淡苔白腻+脉沉弱 |
| **核心病机** | 太阴脾虚寒湿 |
| **核心药证** | 人参→气虚；白术→湿象；干姜→寒象；甘草→调和 |
| **加减原则** | 有证则加，无证则减/调整 |
| **禁忌** | 热证、阴虚、真热假寒 |

**记住**：理中汤是太阴脾虚寒的主方，用前必辨寒热虚实，用药必校验药证。
