# 五苓散药证详解

> 五苓散为太阳蓄水主方，利水渗湿、温阳化气。

---

## 一、《伤寒论》原文

> "太阳病，发汗后，大汗出，胃中干，烦躁不得眠，欲得饮水者，少少与饮之，令胃气和则愈。若脉浮，小便不利，微热消渴者，五苓散主之。"（71条）

> "发汗已，脉浮数，烦渴者，五苓散主之。"（72条）

> "伤寒汗出而渴者，五苓散主之。"（73条）

---

## 二、方剂组成

| 药味 | 剂量（原方） | 功效 | 归经 |
|------|------------|------|------|
| 猪苓 | 十八铢（去皮） | 利水渗湿 | 肾、膀胱 |
| 泽泻 | 一两六铢 | 利水渗湿 | 肾、膀胱 |
| 白术 | 十八铢 | 健脾燥湿 | 脾、胃 |
| 茯苓 | 十八铢 | 利水渗湿 | 心、脾、肾 |
| 桂枝 | 半两（去皮） | 温阳化气 | 心、肺、膀胱 |

**制法**：上五味，为末，以白饮和服方寸匕，日三服。多饮暖水，汗出愈。

---

## 三、方证要点

### 核心病机

太阳蓄水——表邪入里，膀胱气化不利，水饮内停。

### 方证对照表

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 小便不利 | 尿少、尿频 | 膀胱气化不利 |
| 消渴 | 口渴多饮 | 水饮内停，津不上承 |
| 烦渴 | 烦躁口渴 | 水停气化不利 |
| 渴欲饮水 | 想喝水 | 水饮内停 |
| 水入则吐 | 喝水后呕吐 | 水饮内停，胃失和降 |
| 脉浮 | 脉浮 | 表证未解 |
| 微热 | 低热 | 表邪未尽 |
| 或脐下悸 | 小腹跳动 | 水气上逆 |
| 或吐涎沫 | 呕吐清水 | 水饮犯胃 |

### 方证匹配度评估

| 匹配症状数 | 可靠性 | 建议 |
|-----------|--------|------|
| 小便不利+渴+脉浮 | 高 | 方证匹配，可用 |
| 无渴 | — | 考虑他方 |

---

## 四、药证校验

### 泽泻——利水渗湿（主药）

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 利水 | 小便不利 | 水饮内停 ✓ |

**泽泻药证**：凡见小便不利者，泽泻有据。

### 猪苓、茯苓——利水渗湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 利水 | 小便不利 | 水饮内停 ✓ |
| 渗湿 | 水肿 | 水湿内停 ✓ |

### 白术——健脾燥湿

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 健脾 | 食少 | 脾虚 ✓ |
| 燥湿 | 水饮内停 | 湿邪 ✓ |

### 桂枝——温阳化气

| 功效 | 对应症状 | 校验要点 |
|------|---------|---------|
| 温阳 | 小便不利 | 气化不利 ✓ |
| 化气 | 渴 | 津不上承 ✓ |
| 解表 | 微热、脉浮 | 表邪未尽 ✓ |

**桂枝药证**：凡见小便不利+表证者，桂枝有据。

---

## 五、加减法

| 加减方 | 组成变化 | 适用 |
|-------|---------|------|
| 茵陈五苓散 | +茵陈 | 黄疸+水停 |
| 四苓散 | 去桂枝 | 无表证之水停 |

---

## 六、禁忌

| 禁忌情况 | 原因 | 替代方案 |
|---------|------|---------|
| 阴虚津亏 | 利水伤阴 | 慎用或配养阴药 |
| 无水饮征象 | 方证不符 | 重新辨证 |

---

## 七、典型案例

### 案例：太阳蓄水

**患者**：男，50岁

**主诉**：小便不利、口渴3天

**十问**：
- 吃：正常
- 喝：口渴明显，喝水后想吐
- 拉：正常
- 撒：尿少
- 睡：烦躁
- 汗：汗出
- 痛：无
- 寒热：低热

**舌脉**：苔白，脉浮

**辨证**：太阳蓄水

**方证对勘**：
| 方证 | 患者 | 匹配 |
|------|------|------|
| 小便不利 | 尿少 ✓ | ✓ |
| 消渴 | 口渴明显 ✓ | ✓ |
| 水入则吐 | 喝水后想吐 ✓ | ✓ |
| 微热 | 低热 ✓ | ✓ |
| 脉浮 | 脉浮 ✓ | ✓ |

**处方**：五苓散（散剂）
- 猪苓 10g
- 泽泻 15g
- 白术 10g
- 茯苓 10g
- 桂枝 5g

**医嘱**：白饮和服，多饮暖水。

## 八、方剂对比

### 五苓散 vs 真武汤

| 对比项 | 五苓散 | 真武汤 |
|-------|-------|-------|
| 病位 | 太阳（膀胱） | 少阴（肾） |
| 病机 | 气化不利 | 阳虚水停 |
| 表证 | 有（脉浮） | 无 |
| 附子 | 无 | 有 |
| 泽泻猪苓 | 有 | 无 |

**鉴别要点**：有无表证（脉浮、微热）。

### 五苓散 vs 猪苓汤

| 对比项 | 五苓散 | 猪苓汤 |
|-------|-------|-------|
| 病机 | 寒饮内停 | 水热互结 |
| 寒热 | 寒 | 热 |
| 滑石阿胶 | 无 | 有 |
| 桂枝 | 有 | 无 |

**鉴别要点**：寒（苔白）vs 热（舌红、心烦）。

---

## 九、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 小便不利+消渴+脉浮（+水入则吐） |
| **核心病机** | 太阳蓄水（膀胱气化不利） |
| **核心药证** | 泽泻猪苓茯苓→利水；桂枝→化气解表 |
| **特点** | 利水+化气+解表 |
| **服法** | 散剂，白饮和服，多饮暖水 |

**记住**：五苓散治太阳蓄水，关键是"小便不利+渴+脉浮"。有表证是其特点。
