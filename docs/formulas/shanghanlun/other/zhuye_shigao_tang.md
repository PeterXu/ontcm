# 竹叶石膏汤药证详解

> 竹叶石膏汤治余热未清，清热生津。

---

## 一、《伤寒论》原文

> "伤寒解后，虚羸少气，气逆欲吐者，竹叶石膏汤主之。"（397条）

---

## 二、方剂组成

| 药味 | 功效 |
|------|------|
| 竹叶 | 清热除烦 |
| 石膏 | 清热生津 |
| 半夏 | 降逆止呕 |
| 麦门冬 | 滋阴生津 |
| 人参 | 补气 |
| 甘草 | 调和 |
| 粳米 | 护胃 |

---

## 三、方证要点

| 方证要点 | 临床表现 | 医理 |
|---------|---------|------|
| 虚羸少气 | 体虚气短 | 气阴两虚 |
| 气逆欲吐 | 恶心想吐 | 胃气上逆 |

---

## 四、使用总结

| 项目 | 内容 |
|------|------|
| **核心方证** | 虚羸少气+气逆欲吐 |
| **核心病机** | 余热未清气阴两虚 |
| **特点** | 清热生津 |
