# 中医六经辨证开方指南

## 前言

本指南基于六经辨证体系，提供一套系统化的问诊、辨证、选方流程。核心原则：
- **有是证用是方**：每个诊断都要有证据支撑
- **双向验证**：既找支持证据，也找矛盾点
- **层层校验**：问诊→舌脉→定经→方证→药证，每一步都是验证

---

## 第一步：主诉与病史

### 为什么这一步不能跳过
患者最想解决的问题，是整个辨证的"锚点"。没有主诉，容易"漏主病"。

### 问什么

| 维度 | 具体内容 | 为什么重要 |
|------|---------|-----------|
| **主诉** | 患者最想解决什么问题？ | 确定辨证主攻方向 |
| **现病史** | 发病多久？什么原因诱发？加重/缓解因素？ | 判断病程长短、病势缓急 |
| **治疗史** | 之前吃过什么药？效果如何？ | 避免重复无效方案 |
| **既往史** | 有无慢性病、手术史、过敏史？ | 排除禁忌，判断体质 |

### 示例
> 主诉：腹泻3天
> 现病史：3天前吃生冷后开始腹泻，日行5-6次，水样便，伴腹痛肠鸣，怕冷，食欲差
> 治疗：自服黄连素无效
> 既往：体健，无特殊病史

**记录要点**：用患者原话记录，不要过早用自己的术语概括。

---

## 第一步后：急危重症排除 ⚠️

> 辨证前，先排除需要紧急就医的情况。中医辨证适用于慢性病、轻中度急性病，急危重症需中西医结合处理。

### 需紧急就医的信号

| 表现 | 可能情况 | 建议 |
|------|---------|------|
| 持续剧烈腹痛拒按 | 急腹症（阑尾炎、穿孔、肠梗阻等） | 立即就医，排除外科情况 |
| 高热不退伴神志改变 | 重症感染、脑膜炎、脑炎 | 立即就医 |
| 突发剧烈胸痛伴大汗、气短 | 心梗可能 | 立即就医，拨打急救电话 |
| 突发剧烈头痛、呕吐 | 脑出血、脑卒中可能 | 立即就医 |
| 呼吸困难、喘促不止 | 重症肺炎、哮喘发作、肺栓塞 | 立即就医 |
| 呕血、黑便（柏油样便） | 上消化道出血 | 立即就医 |
| 持续腹泻伴严重脱水表现（口干舌燥、皮肤弹性差、血压下降） | 严重脱水、休克风险 | 立即就医补液 |
| 少阴危候：但欲寐、肢冷、脉微欲绝、冷汗淋漓 | 亡阳危候 | 急用四逆汤抢救，同时送医 |
| 意识模糊、抽搐 | 颅内病变、严重电解质紊乱 | 立即就医 |

### 排查方法

在记录主诉后，问自己：

1. **这个症状是否可能是急危重症的表现？**
2. **患者的精神状态如何？是否萎靡、烦躁、嗜睡？**
3. **生命体征是否平稳？（呼吸、脉搏、血压）**
4. **是否需要先做实验室检查或影像学检查排除器质性病变？**

### 中医适用范围界定

| 适用 | 不适用（需先西医排查） |
|------|----------------------|
| 慢性病调理 | 急腹症、外伤 |
| 轻中度急性病（感冒、腹泻、胃炎等） | 重症感染、器官衰竭 |
| 功能性疾病 | 器质性病变（肿瘤、梗死等） |
| 亚健康状态 | 急危重症 |

**原则**：
- 中医辨证**不替代**西医诊断
- 遇到急危重症信号，**先排除危险情况**，再进行辨证
- 不确定时，建议患者先做相关检查

## 第二步：十问为纲

传统"十问歌"是问诊的框架，本指南以"吃喝拉撒睡汗痛"为核心，辅以其他维度。

### 2.1 吃：脾胃的底牌

> 问什么：饭量、食欲、口味偏好、饭后反应

一个"吃"字，能问出太阴、阳明、少阳三条经的信息。

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 能吃但吃完胀 | 太阴 | 脾虚不运 |
| 不想吃、没胃口 | 太阴为主 | 脾胃虚弱或湿阻 |
| 饿得快但吃了难受 | 阳明+太阴 | 胃强脾弱 |
| 口苦 | 少阳 | 胆火上炎 |
| 口淡 | 太阴 | 脾虚湿盛 |
| 口干想喝凉水 | 阳明 | 阳明热盛 |
| 口干不想喝水 | 少阴/太阴 | 阴证，阳气不化 |

### 2.2 喝：津液和寒热的晴雨表

> 问什么：渴不渴、想喝什么温度的水、喝多少

"喝"这个维度，核心看的是寒热。寒热一定，大方向就定了。

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 口渴想喝大量凉水 | 阳明 | 阳明热盛 |
| 口渴但不想喝 | 太阴/少阴 | 水饮内停或阳气不化 |
| 口不渴 | 太阴/少阴 | 寒证为主 |
| 想喝热水 | 太阴/少阴 | 里寒 |

### 2.3 拉：气机升降的出口

> 问什么：大便次数、形状、颜色、费力程度、有无粘液脓血

大便告诉你的不只是肠道的问题，是整个气机升降的状态。

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 便秘、干硬 | 阳明 | 阳明腑实 |
| 便秘但不干、拉得费劲 | 太阴/少阴 | 气虚或阳虚推动无力 |
| 大便稀、不成形 | 太阴 | 脾虚湿盛 |
| 大便先干后稀 | 太阴 | 脾虚兼有积滞 |
| 水样便、拉得凶 | 太阴/少阴 | 太阴寒湿或少阴下利 |
| 粘液脓血、里急后重 | 阳明/厥阴 | 湿热痢疾 |

### 2.4 撒：水液代谢的窗口

> 问什么：小便次数、量、颜色、有无不适（涩痛、灼热）、夜尿次数

小便和喝水配合着看，水液代谢的全貌就出来了。

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 尿频尿急 | 下焦热/肾阳虚 | 膀胱有热或阳虚不固 |
| 尿少色黄 | 阳明/少阳 | 里热伤津 |
| 尿清长 | 太阴/少阴 | 寒证 |
| 夜尿多 | 少阴 | 肾阳虚 |
| 尿涩痛 | 太阳/阳明 | 湿热下注（淋证） |

### 2.5 睡：阴阳交会的镜子

> 问什么：入睡难不难、睡得深不深、几点醒、梦多不多

睡眠是阴阳状态最直接的反映，问诊一定不能跳过。

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 入睡困难 | 少阳/阳明 | 阳不入阴，少阳郁热或阳明热扰 |
| 易醒、多梦 | 少阳/心血不足 | 肝胆火旺或血不养心 |
| 凌晨1-3点醒 | 厥阴 | 肝经问题 |
| 凌晨3-5点醒 | 太阴 | 肺经问题 |
| 嗜睡、睡不够 | 太阴/少阴 | 阳虚或湿困 |
| 但欲寐 | 少阴 | 少阴阳虚 |

### 2.6 汗：表里证的关键

> 问什么：出汗与否、出汗时间、部位、多少

**⚠️ 这是传统问诊第一要问的内容，不可省略。**

| 表现 | 辨证指向 | 病机 |
|------|---------|------|
| 无汗、恶寒 | 太阳表实 | 寒邪束表 |
| 有汗、恶风 | 太阳表虚 | 营卫不和 |
| 但头汗出 | 少阳 | 少阳郁热 |
| 大汗、大渴 | 阳明 | 阳明热盛 |
| 自汗（白天出汗） | 太阳/太阴 | 气虚不固 |
| 盗汗（夜间出汗） | 少阴/阴虚 | 阴虚内热 |
| 冷汗淋漓 | 少阴 | 亡阳 |

### 2.7 痛：经络定位的线索

> 问什么：疼痛部位、性质（胀痛/刺痛/隐痛/绞痛）、加重/缓解因素

疼痛部位直接对应经络，是辨证的重要依据。

| 疼痛部位 | 辨证指向 | 备注 |
|---------|---------|------|
| 头痛（后脑） | 太阳 | 太阳经循行部位 |
| 头痛（前额） | 阳明 | 阳明经循行部位 |
| 头痛（两侧） | 少阳 | 少阳经循行部位 |
| 头痛（巅顶） | 厥阴 | 厥阴经循行部位 |
| 胸胁胀满/疼痛 | 少阳 | 少阳经循行部位 |
| 少腹冷痛 | 厥阴/少阴 | 寒凝肝脉或肾阳虚 |
| 腰痛 | 少阴 | 肾主腰 |

**疼痛性质辨虚实**：
- 胀痛 → 气滞
- 刺痛 → 瘀血
- 隐痛 → 虚证
- 绞痛 → 实证/寒凝

### 2.8 其他重要问诊

#### 寒热
> 问什么：怕冷还是怕热、有无发热、发热类型

| 表现 | 辨证指向 |
|------|---------|
| 恶寒发热同时出现 | 太阳表证 |
| 但热不寒 | 阳明里热 |
| 寒热往来 | 少阳 |
| 但寒不热 | 少阴/太阴里寒 |

#### 情志
> 问什么：烦躁、抑郁、焦虑、易怒

| 表现 | 辨证指向 |
|------|---------|
| 烦躁 | 阳明热扰或少阴阴虚 |
| 抑郁、善太息 | 少阳肝郁 |
| 易怒 | 厥阴/少阳肝火 |

#### 女性：经带问诊
> 问什么：月经周期、量、色、质、有无痛经；带下颜色、气味、量

| 表现 | 辨证指向 |
|------|---------|
| 月经后期、色淡量少 | 太阴血虚 |
| 月经前期、色红量多 | 阳明血热 |
| 痛经、经血有块 | 厥阴瘀血 |
| 带下清稀 | 太阴/少阴寒湿 |
| 带下黄稠 | 阳明/少阳湿热 |

## 第三步：舌诊

> 舌诊是望诊的核心，提供客观的寒热、虚实、湿邪信息。

### 观察维度

| 维度 | 正常 | 异常表现及意义 |
|------|------|--------------|
| **舌质颜色** | 淡红 | 淡白→气血两虚/阳虚；红→热；绛→热入营血；紫→瘀血 |
| **舌体形态** | 大小适中、无齿痕 | 胖大有齿痕→太阴脾虚湿盛；瘦薄→阴虚；裂纹→阴液亏损 |
| **舌苔颜色** | 薄白 | 白→寒/湿；黄→热；灰黑→寒极或热极 |
| **舌苔厚度** | 薄 | 厚→邪气盛；腻→湿浊；燥→津液伤 |
| **舌苔分布** | 均匀 | 根部厚腻→下焦湿热；中部厚腻→中焦湿阻 |

### 常见舌象速查

| 舌象 | 辨证指向 | 典型证型 |
|------|---------|---------|
| 舌淡苔白腻 | 太阴 | 脾虚湿盛 |
| 舌红苔黄燥 | 阳明 | 阳明腑实 |
| 舌红苔黄腻 | 阳明/少阳 | 湿热蕴结 |
| 舌淡胖齿痕 | 太阴/少阴 | 脾肾阳虚 |
| 舌绛无苔 | 少阴/厥阴 | 阴虚火旺或热入营血 |
| 舌紫暗有瘀点 | 厥阴 | 瘀血阻滞 |

## 第四步：脉诊

> 脉诊是辨证的硬验证。问诊是患者的自我感受，脉诊是客观体征对你的判断说"是"还是"否"。

### 常见脉象与六经对应

| 脉象 | 特征 | 辨证指向 | 典型证型 |
|------|------|---------|---------|
| 浮脉 | 轻取即得 | 太阳 | 表证 |
| 浮紧 | 浮而紧绷 | 太阳表实 | 麻黄汤证 |
| 浮缓 | 浮而松弛 | 太阳表虚 | 桂枝汤证 |
| 沉脉 | 重按始得 | 少阴/太阴 | 里证 |
| 沉弱 | 沉而无力 | 少阴/太阴 | 阳虚 |
| 沉迟 | 沉而缓慢 | 少阴/太阴 | 里寒 |
| 数脉 | 一息六至以上 | 阳明/少阳 | 热证 |
| 洪脉 | 来盛去衰 | 阳明 | 阳明经热（白虎汤证） |
| 滑脉 | 往来流利 | 阳明/痰湿 | 痰热、食积 |
| 弦脉 | 如按琴弦 | 少阳/厥阴 | 肝郁、气滞 |
| 细脉 | 脉细如线 | 少阴/厥阴 | 血虚、阴虚 |
| 微脉 | 极细极软 | 少阴 | 亡阳 |

### 脉证合参

**脉和判断一致** → 信心加倍
**脉和判断矛盾** → 必须回头重新审视

例如：
- 判断阳明热盛 → 脉应该是洪大或滑数。如果脉反而细弱，重新考虑。
- 判断少阴阳虚 → 脉应该是微细或沉迟。如果脉浮数，方向可能错了。
- 判断太阳表实 → 脉应该是浮紧。如果脉浮缓，可能是桂枝汤证而非麻黄汤证。

## 第五步：定经

> 把问诊、舌诊、脉诊的信息往六经上归。这一步确定大方向。

### 八纲为纲：阴阳、表里、寒热、虚实

#### 5.1 先定表里

| 分类 | 表现 | 六经归属 |
|------|------|---------|
| 表证 | 恶寒发热、头痛身痛、脉浮 | 太阳 |
| 里证 | 但里无表，吃喝拉撒睡的问题为主 | 阳明、太阴、少阴 |
| 半表半里 | 忽冷忽热、口苦、胸胁苦满 | 少阳 |

#### 5.2 再定寒热

| 分类 | 表现 | 六经归属 |
|------|------|---------|
| 热证 | 口渴喜凉、尿黄、舌红、脉数 | 阳明 |
| 寒证 | 口不渴或渴喜热、尿清、舌淡、脉迟 | 太阴、少阴 |
| 寒热错杂 | 上热下寒、口苦但便稀 | 厥阴 |

#### 5.3 定虚实

| 分类 | 表现 | 病机 |
|------|------|------|
| 实证 | 声高气粗、痛拒按、脉有力 | 邪气盛 |
| 虚证 | 声低气怯、痛喜按、脉无力 | 正气虚 |

#### 5.4 最后定经

把表里、寒热、虚实和前面问诊、舌诊、脉诊的结果叠在一起，六经就出来了。

### 六经证候要点速查

| 六经 | 核心病机 | 典型表现 | 代表方剂 |
|------|---------|---------|---------|
| 太阳 | 表寒 | 恶寒发热、头痛、脉浮 | 麻黄汤、桂枝汤 |
| 阳明 | 里热 | 大热大汗大渴、脉洪大 | 白虎汤、承气汤 |
| 少阳 | 半表半里、枢机不利 | 往来寒热、口苦、胸胁苦满、默默不欲饮食 | 小柴胡汤 |
| 太阴 | 里寒湿、脾虚 | 腹满而吐、自利不渴、脉沉弱 | 理中汤 |
| 少阴 | 里虚寒或里虚热 | 但欲寐、脉微细 | 四逆汤、黄连阿胶汤 |
| 厥阴 | 寒热错杂 | 消渴、气上撞心、心中疼热、饥而不欲食 | 乌梅丸 |

## 第六步：方证对勘

> 患者的表现，和这个方的方证对得上吗？这是方药层面的第一重校验。

### 方证匹配原则

1. **核心症状必须匹配**：每个方剂都有核心症状，必须具备
2. **症状越多越稳**：方证对得越多，判断越可靠
3. **对不上要反思**：对得少，想想是合病？还是辨错了？

### 常用方剂方证速查（附《伤寒论》原文）

#### 麻黄汤证

**《伤寒论》原文**：
> "太阳病，头痛发热，身疼腰痛，骨节疼痛，恶风无汗而喘者，麻黄汤主之。"（35条）

**方证要点**：
- 恶寒、无汗、身痛 ✓
- 脉浮紧 ✓

| 方剂 | 核心病机 | 核心症状 | 脉象 |
|------|---------|---------|------|
| 麻黄汤 | 太阳表实 | 发热、恶寒、无汗、身痛、喘 | 浮紧 |

#### 桂枝汤证

**《伤寒论》原文**：
> "太阳病，头痛，发热，汗出，恶风，桂枝汤主之。"（13条）
> "太阳病，初服桂枝汤，反烦不解者，先刺风池、风府，却与桂枝汤则愈。"（24条）

**方证要点**：
- 发热、恶风、汗出 ✓
- 脉浮缓 ✓

| 方剂 | 核心病机 | 核心症状 | 脉象 |
|------|---------|---------|------|
| 桂枝汤 | 太阳表虚 | 发热、恶风、汗出、头痛 | 浮缓 |

**关键鉴别点**：有汗无汗

#### 白虎汤证

**《伤寒论》原文**：
> "伤寒脉浮滑，此以表有热，里有寒，白虎汤主之。"（176条）
> "三阳合病，腹满身重，难以转侧，口不仁面垢，谵语遗尿，发汗则谵语甚，下之则额上生汗，手足逆冷。若自汗出者，白虎汤主之。"（219条）

**方证要点**：
- 大热、大汗、大渴、脉洪大 ✓（"四大"）
- 舌红苔黄燥 ✓

| 方剂 | 核心病机 | 核心症状 | 舌象 | 脉象 |
|------|---------|---------|------|------|
| 白虎汤 | 阳明经热（无形） | 大热、大汗、大渴、脉洪大 | 舌红苔黄燥 | 洪大 |

#### 承气汤证

**《伤寒论》原文**：
> "阳明病，谵语，有潮热，反不能食者，胃中必有燥屎五六枚也；若能食者，但硬尔。宜大承气汤下之。"（215条）
> "伤寒若吐若下后不解，不大便五六日，上至十余日，日晡所发潮热，不恶寒，独语如见鬼状；若剧者，发则不识人，循衣摸床，惕而不安，微喘直视，脉弦者生，涩者死。微者，但发热谵语者，大承气汤主之。"（212条）

**方证要点**：
- 日晡潮热、腹满痛拒按、大便不通 ✓
- 谵语、苔焦黑 ✓

| 方剂 | 核心病机 | 核心症状 | 舌象 | 脉象 |
|------|---------|---------|------|------|
| 承气汤 | 阳明腑实（有形） | 日晡潮热、腹满痛拒按、大便不通、谵语 | 舌红苔黄燥或焦黑 | 沉实 |

**关键鉴别点**：有无腑实（腹满痛、大便不通）

#### 小柴胡汤证

**《伤寒论》原文**：
> "伤寒五六日中风，往来寒热，胸胁苦满，嘿嘿不欲饮食，心烦喜呕，或胸中烦而不呕，或渴，或腹中痛，或胁下痞鞕，或心下悸、小便不利，或不渴、身有微热，或咳者，小柴胡汤主之。"（96条）
> "但见一证便是，不必悉具。"（101条）

**方证要点**（但见一证便是，不必悉具）：
- 往来寒热 ✓
- 胸胁苦满 ✓
- 默默不欲饮食 ✓
- 心烦喜呕 ✓
- 口苦、咽干、目眩 ✓

#### 理中汤证

**《伤寒论》原文**：
> "霍乱，头痛发热，身疼痛，热多欲饮水者，五苓散主之；寒多不用水者，理中汤主之。"（386条）
> "胸痹心中痞，留气结在胸，胸满，胁下逆抢心，枳实薤白桂枝汤主之；人参汤亦主之。"（人参汤即理中汤）

**方证要点**：
- 腹满、吐利 ✓
- 口不渴、喜温 ✓
- 舌淡苔白腻 ✓

#### 四逆汤证

**《伤寒论》原文**：
> "少阴病，脉微细，但欲寐也。"（281条）
> "少阴病，脉沉者，急温之，宜四逆汤。"（323条）
> "少阴病，下利清谷，里寒外热，手足厥逆，脉微欲绝，身反不恶寒，其人面色赤，或腹痛，或干呕，或咽痛，或利止脉不出者，通脉四逆汤主之。"（317条）

**方证要点**：
- 但欲寐、精神萎靡 ✓
- 肢冷、下利清谷 ✓
- 脉微细或微欲绝 ✓

## 第七步：药证校验

> 方证对了大方向，药证是进一步精细校验——选的这个方子里，每味药在这个患者身上有没有依据？

### 药证校验方法

逐味药问自己：这味药在这个患者身上有对应的症状吗？

### 示例：太阴脾虚寒，选理中汤

理中汤四味药：人参、白术、干姜、炙甘草

| 药味 | 功效 | 对应症状 | 校验结果 |
|------|------|---------|---------|
| 人参 | 补气 | 乏力、少气懒言、脉弱 | ✓ 有依据 |
| 白术 | 健脾燥湿 | 便稀、苔白腻 | ✓ 有依据 |
| 干姜 | 温中散寒 | 口淡、喜热饮、四肢凉 | ✓ 有依据 |
| 炙甘草 | 调和诸药、补中益气 | — | ✓ 辅助用药 |

**四味药都有症状支撑 → 这个方用着踏实。**

### 加减原则

**加减不是凭感觉，是凭证据。**

| 情况 | 操作 | 依据 |
|------|------|------|
| 症状多一味药的功效 | 加药 | 如理中汤证兼呕吐，加半夏降逆 |
| 症状少一味药的功效 | 减药或换药 | 如便不稀，白术可减量或炒用减燥性 |
| 有新症状 | 加新药 | 如兼表证，加桂枝解表 |
| 病机变化 | 换方 | 药证对不上了，说明辨证要调整 |

---

## 第八步：证据核查（证明）

> 把支持这个诊断的证据全列出来，看看够不够硬。

### 操作方法

列出诊断后，逐条写明支持证据：

**示例：诊断"太阴脾虚寒，理中汤证"**

| 证据 | 支持诊断 |
|------|---------|
| 便稀 | ✓ |
| 口淡不渴 | ✓ |
| 食欲差 | ✓ |
| 四肢凉 | ✓ |
| 脉沉弱 | ✓ |
| 舌淡苔白 | ✓ |

**六个证据全支持 → 诊断很稳。**

### 判断标准

- 证据充分（≥5条支持）→ 诊断可靠
- 证据一般（3-4条支持）→ 继续观察，注意合病可能
- 证据不足（<3条支持）→ 考虑是否有其他可能，是否辨错了

---

## 第九步：反向验证（证伪）

> 证明是确认你对在哪，证伪是检查你错在哪。这是最重要的一步。

### 操作方法

列出诊断后，**主动寻找矛盾点**：

**示例：诊断"太阴脾虚寒"后，寻找矛盾症状**

| 症状 | 与诊断的矛盾 | 可能提示 |
|------|------------|---------|
| 患者说口苦 | 太阴不应该口苦，少阳才有口苦 | 是否有少阳合并？ |
| 偶尔怕冷发热同时出现 | 不像纯里证 | 是否有表证未解？ |
| 脉虽然沉，但偏数 | 寒证不应该脉数 | 是否有化热趋势？ |

### 发现矛盾点后的处理

| 矛盾类型 | 处理方式 |
|---------|---------|
| 症状矛盾 | 考虑合病（如太阴+少阳） |
| 脉证矛盾 | 以脉为重，重新审视问诊 |
| 舌脉矛盾 | 结合病机判断，舌象反映体质，脉象反映当前病势 |

**证伪不是否定自己，是让诊断更完整、更经得起检验。**

---

## 第十步：鉴别诊断

> 与相似证型逐条对比，排除其他可能。

### 常见鉴别

#### 太阴 vs 少阴

| 鉴别点 | 太阴 | 少阴 |
|-------|------|------|
| 病位 | 脾 | 肾 |
| 核心症状 | 腹满、吐利 | 但欲寐、肢冷 |
| 脉象 | 沉弱 | 微细 |
| 精神状态 | 神志清楚 | 精神萎靡 |
| 病势 | 较缓 | 较急，有亡阳风险 |

#### 太阳表实 vs 太阳表虚

| 鉴别点 | 麻黄汤证（表实） | 桂枝汤证（表虚） |
|-------|----------------|----------------|
| 汗出 | 无汗 | 有汗 |
| 恶寒程度 | 重 | 轻，恶风为主 |
| 疼痛 | 身痛明显 | 身痛较轻 |
| 脉象 | 浮紧 | 浮缓 |

#### 阳明经证 vs 阳明腑证

| 鉴别点 | 白虎汤证（经证） | 承气汤证（腑证） |
|-------|----------------|----------------|
| 腹部体征 | 无腹胀痛 | 腹满痛拒按 |
| 大便 | 可能正常 | 便秘或热结旁流 |
| 神志 | 清楚 | 可有谵语 |
| 舌苔 | 黄燥 | 焦黑起刺 |

## 第十一步：合病并病排查

> 实际临证中，单一经病并不常见，要警惕合病、并病。

### 合病

**定义**：两经或三经同时发病

**常见类型**：
| 合病 | 表现 | 选方思路 |
|------|------|---------|
| 太阳阳明合病 | 发热恶寒+口渴汗出 | 葛根汤、麻杏石甘汤等 |
| 太阳少阳合病 | 发热恶寒+口苦胸胁苦满 | 柴胡桂枝汤 |
| 少阳阳明合病 | 口苦胸胁苦满+便秘 | 大柴胡汤 |

### 并病

**定义**：一经病未愈，又传另一经

**处理原则**：
- 表里同病：先表后里（特殊情况先里后表）
- 虚实夹杂：扶正祛邪并用

### 排查方法

在完成前述辨证后，问自己：
1. 患者的症状能否用一经病完全解释？
2. 如果不能，是否有两经或三经同时发病？
3. 是否有一经病未愈，又出现新的经病表现？

## 第十二步：选方定药

> 走完前面十一步，选方已经不是"猜"了，是水到渠成。

### 选方原则

1. **主方确定**：根据六经辨证和方证对勘
2. **加减依据**：根据药证校验和患者个体情况
3. **剂量原则**：体质强弱、病情轻重、年龄长幼

### 剂量考量

| 因素 | 剂量调整 |
|------|---------|
| 体质强壮 | 可用常量或稍重 |
| 体质虚弱 | 减量，中病即止 |
| 老人小儿 | 减量 |
| 病情急重 | 顿服、急煎 |
| 病情缓轻 | 常规煎服 |

### 煎服法

| 类型 | 煎法 | 服法 |
|------|------|------|
| 解表剂 | 武火急煎，香气出即止 | 温服，覆被取微汗 |
| 补益剂 | 文火慢煎 | 空腹或饭前服 |
| 泻下剂 | 常规煎煮 | 饭后服，中病即止 |
| 调和剂 | 常规煎煮 | 不拘时服 |

### 饮食禁忌

| 证型 | 禁忌 |
|------|------|
| 表证 | 生冷、油腻、粘滑 |
| 热证 | 辛辣、温燥 |
| 寒证 | 生冷、瓜果 |
| 脾胃病 | 生冷、油腻、难消化 |

### 用方禁忌速查 ⚠️

> 选方前必须排查禁忌，避免不良反应。

| 方剂 | 禁忌人群/情况 | 原因 | 替代方案 |
|------|-------------|------|---------|
| **麻黄汤** | 高血压患者 | 麻黄升压、兴奋心脏 | 改用荆防败毒散等温和解表方 |
| **麻黄汤** | 心脏病、心律失常患者 | 麻黄兴奋心脏，加重心律问题 | 改用温和解表方 |
| **麻黄汤** | 体虚、老人、孕妇 | 发汗太过伤正气 | 改用桂枝汤或参苏饮 |
| **麻黄汤** | 失眠、焦虑患者 | 麻黄兴奋中枢 | 改用温和解表方 |
| **桂枝汤** | 表实无汗者（非禁忌但不对证） | 方证不符，需辨表实表虚 | 无汗者用麻黄汤 |
| **承气汤类** | 孕妇 | 攻下伤胎，可能流产 | 改用润下或缓下 |
| **承气汤类** | 体虚者、老人 | 攻下伤正，正气更虚 | 改用增液承气或润下 |
| **承气汤类** | 产后、经期 | 攻下伤气血 | 改用温和通便法 |
| **四逆汤** | 热证假寒（真热假寒） | 热证用热药加重病情 | 需先辨清寒热真假 |
| **四逆汤** | 阴虚发热 | 温药加重阴虚 | 改用养阴清热方 |
| **白虎汤** | 表证未解者 | 清里过早，引邪入里 | 先解表，表解再清里 |
| **白虎汤** | 无汗者（阳明经热应有汗） | 方证不符 | 需辨证是否阳明 |
| **小柴胡汤** | 纯表证或纯里证 | 方证不符（半表半里方） | 辨证后再选方 |

### 特殊人群用药注意

| 人群 | 注意事项 |
|------|---------|
| **孕妇** | 禁用攻下、破血、峻下逐水药；慎用活血、滑利药 |
| **老人** | 剂量宜轻，攻伐药慎用，扶正为主 |
| **小儿** | 剂量按体重计算，药性宜平和 |
| **体虚者** | 攻伐药减量或配扶正药，中病即止 |

## 附录一：六经传变规律

> 六经辨证不是静态的，病情会演变。理解传变规律，才能预判病情发展。

### 传变方式

| 传变类型 | 说明 | 示例 |
|---------|------|------|
| **顺传** | 太阳→阳明→少阳→太阴→少阴→厥阴 | 病情由表入里，由轻转重 |
| **越传（直中）** | 越过某些经，直接传入 | 太阳直入少阴（体虚者）；表邪直入太阴 |
| **合病** | 两三经同时发病 | 太阳阳明合病、太阳少阳合病 |
| **并病** | 一经未罢又传他经 | 太阳并病少阳 |
| **逆转（向愈）** | 里证转表证，或症状减轻 | 太阴转太阳（脾虚恢复，表证出现） |

### 判断传变的方法

问自己：

1. **患者症状是否在演变？**
   - 新症状出现？
   - 旧症状消失或加重？

2. **当前在哪一经？是否在向下一经发展？**
   - 太阳 → 恶寒减轻、出现口渴 → 可能传入阳明
   - 太阴 → 精神萎靡、肢冷加重 → 可能传入少阴

3. **是否有逆转趋势？**
   - 里证转表证 = 向愈信号
   - 腹泻减轻、食欲恢复 = 太阴向愈

### 传变的影响因素

| 因素 | 影响 |
|------|------|
| **体质强弱** | 体强者顺传，体弱者易直中少阴 |
| **治疗得当** | 可阻断传变，逆转向愈 |
| **治疗不当** | 加速传变，引邪入里 |
| **误汗/误下** | 太阳误下→传入太阴；阳明误汗→伤津化燥 |

### 临床意义

| 情况 | 处理 |
|------|------|
| 发现传变趋势 | 提前干预，阻断传变 |
| 已传变 | 及时调整辨证和方药 |
| 向愈逆转 | 效不更方，继续巩固 |
| 传变不明 | 密切观察，动态评估 |

## 附录二：完整案例演示

### 案例：腹泻3天

#### 第一步：主诉与病史
- **主诉**：腹泻3天
- **现病史**：3天前吃生冷后开始腹泻，日行5-6次，水样便，伴腹痛肠鸣，怕冷，食欲差
- **治疗**：自服黄连素无效
- **既往**：体健，无特殊病史

#### 第二步：十问为纲
- **吃**：食欲差，不想吃，口淡
- **喝**：不想喝水，想喝热水
- **拉**：腹泻，日5-6次，水样便，不臭
- **撒**：尿少色清
- **睡**：尚可
- **汗**：无汗
- **痛**：腹痛肠鸣，喜温喜按
- **寒热**：怕冷，无发热
- **情志**：正常
- **经带**：女，月经正常

#### 第三步：舌诊
舌淡胖，边有齿痕，苔白腻水滑

#### 第四步：脉诊
脉沉弱

#### 第五步：定经
- **表里**：但里无表 → 里证
- **寒热**：口不渴、想喝热水、尿清、舌淡 → 寒证
- **虚实**：脉弱、痛喜按 → 虚证
- **定经**：里寒虚 → 太阴

#### 第六步：方证对勘
**理中汤方证**：
- 腹满而吐 → 有腹痛肠鸣（对应）
- 自利不渴 → 有腹泻、口不渴（对应）
- 霍乱、头痛发热 → 无（非必要症状）
- 舌淡苔白 → 有（对应）

方证匹配度高。

#### 第七步：药证校验
理中汤四味药：
- **人参**：补气 → 患者乏力、脉弱 → ✓
- **白术**：健脾燥湿 → 患者便稀、苔腻 → ✓
- **干姜**：温中散寒 → 患者口淡、喜热、怕冷、四肢凉 → ✓
- **炙甘草**：调和 → 辅助 ✓

四味药都有依据。

#### 第八步：证据核查
诊断"太阴脾虚寒湿，理中汤证"的支持证据：
1. 腹泻水样便 ✓
2. 口淡不渴 ✓
3. 想喝热水 ✓
4. 尿清 ✓
5. 怕冷 ✓
6. 食欲差 ✓
7. 腹痛喜温喜按 ✓
8. 舌淡胖齿痕苔白腻 ✓
9. 脉沉弱 ✓

**证据充分，诊断可靠。**

#### 第九步：反向验证
寻找矛盾症状：
- 有口苦吗？→ 无，太阴不应有口苦 ✓
- 有发热吗？→ 无，纯里证 ✓
- 脉数吗？→ 不数，沉弱，符合寒证 ✓

**未发现矛盾点，诊断成立。**

#### 第十步：鉴别诊断
与相似证型鉴别：
- **少阴下利**：少阴有"但欲寐"、精神萎靡、肢冷脉微。本例患者精神可，无但欲寐 → 排除少阴。
- **阳明热利**：阳明热利大便臭秽、肛门灼热。本例患者大便不臭、无肛门灼热 → 排除阳明。

**确定为太阴脾虚寒湿。**

#### 第十一步：合病排查
症状能否用太阴一经完全解释？能。
无合病表现。

#### 第十二步：选方定药
**处方**：理中汤
- 人参 9g
- 白术 12g（炒用，减燥性）
- 干姜 9g
- 炙甘草 6g

**煎服法**：水煎服，日一剂，分两次温服

**医嘱**：
1. 忌生冷、油腻
2. 饮食宜温热易消化
3. 如腹泻减轻，继续服用；如腹泻加重或出现发热，及时就诊

**复诊反馈**：
3天后复诊，腹泻减为日1-2次，成形便，腹痛消失，食欲恢复。

**处置**：效不更方，继服3剂巩固。

```
定经（表里寒热虚实）→ 方证对勘 → 药证校验
    ↓
证据核查 → 反向验证 → 鉴别诊断 → 合病排查
    ↓
选方定药 → 剂量煎服法 → 医嘱
    ↓
复诊 → 评估 → 调整
```

## 使用说明

1. **初学者**：按十二步顺序执行，不要跳步
2. **有经验者**：可并行执行某些步骤，但所有步骤都要在内心过一遍
3. **核心原则**：
   - 每一步都是验证，不是走形式
   - 发现矛盾立刻回头，不要强行解释
   - 药证、方证、脉证要一一对应

**记住：辨证不是猜，是证据链的闭环。**
