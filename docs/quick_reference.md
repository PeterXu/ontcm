# 中医六经辨证速查表

> 本表为临床快速查阅设计，建议打印随身携带。

---

## 一、六经证候速查

| 六经 | 核心病机 | 典型表现 | 代表方剂 | 舌象 | 脉象 |
|------|---------|---------|---------|------|------|
| **太阳** | 表寒 | 恶寒发热、头痛、身痛 | 麻黄汤（无汗）、桂枝汤（有汗） | 苔薄白 | 浮紧/浮缓 |
| **阳明** | 里热 | 大热大汗大渴、腹满痛 | 白虎汤（经热）、承气汤（腑实） | 舌红苔黄燥 | 洪大/沉实 |
| **少阳** | 半表半里 | 往来寒热、口苦、胸胁苦满 | 小柴胡汤 | 苔薄黄 | 弦 |
| **太阴** | 里寒湿 | 腹满吐利、口不渴 | 理中汤 | 舌淡苔白腻 | 沉弱 |
| **少阴** | 里虚寒/热 | 但欲寐、肢冷脉微 | 四逆汤（寒）、黄连阿胶汤（热） | 舌淡/舌红少苔 | 微细 |
| **厥阴** | 寒热错杂 | 消渴、气上撞心、饥不欲食 | 乌梅丸 | 舌红苔白或黄 | 弦细 |

## 二、太阳病鉴别

### 麻黄汤证 vs 桂枝汤证

| 鉴别点 | 麻黄汤证（表实） | 桂枝汤证（表虚） |
|-------|----------------|----------------|
| **汗出** | 无汗 ✓ | 有汗 ✓ |
| **恶寒程度** | 重 | 轻（恶风） |
| **身痛** | 明显 | 较轻 |
| **脉象** | 浮紧 | 浮缓 |
| **病机** | 寒邪束表，腠理闭塞 | 营卫不和，腠理疏松 |

---

## 三、阳明病鉴别

### 白虎汤证 vs 承气汤证

| 鉴别点 | 白虎汤证（经热） | 承气汤证（腑实） |
|-------|----------------|----------------|
| **腹部** | 无腹胀痛 | 腹满痛拒按 ✓ |
| **大便** | 可能正常 | 便秘或热结旁流 ✓ |
| **神志** | 清楚 | 可有谵语 |
| **舌苔** | 黄燥 | 焦黑起刺 |
| **脉象** | 洪大 | 沉实 |
| **病机** | 热盛伤津（无形） | 燥屎内结（有形） |

---

## 四、太阴 vs 少阴鉴别

| 鉴别点 | 太阴 | 少阴 |
|-------|------|------|
| **病位** | 脾 | 肾 |
| **核心症状** | 腹满、吐利 | 但欲寐、肢冷 |
| **精神状态** | 神志清楚 | 精神萎靡 ✓ |
| **脉象** | 沉弱 | 微细 ✓ |
| **病势** | 较缓 | 较急，有亡阳风险 ✓ |

## 五、常用方剂方证速查

### 解表剂

| 方剂 | 方证要点 | 舌脉 |
|------|---------|------|
| **麻黄汤** | 恶寒、无汗、身痛、脉浮紧 | 苔薄白，脉浮紧 |
| **桂枝汤** | 发热、恶风、汗出、脉浮缓 | 苔薄白，脉浮缓 |
| **葛根汤** | 太阳阳明合病，项背强几几 | 苔薄白，脉浮 |

### 清热剂

| 方剂 | 方证要点 | 舌脉 |
|------|---------|------|
| **白虎汤** | 大热、大汗、大渴、脉洪大 | 舌红苔黄燥，脉洪大 |
| **白虎加人参汤** | 白虎汤证+汗多、脉大无力 | 舌红苔黄燥，脉洪大无力 |

### 攻下剂

| 方剂 | 方证要点 | 舌脉 |
|------|---------|------|
| **大承气汤** | 腹满痛拒按、便秘、谵语、潮热 | 苔焦黑起刺，脉沉实 |
| **小承气汤** | 腹满、便秘、潮热（较轻） | 苔黄厚，脉沉实 |
| **调胃承气汤** | 腹满、便秘、心烦 | 苔黄燥，脉沉 |

### 和解剂

| 方剂 | 方证要点 | 舌脉 |
|------|---------|------|
| **小柴胡汤** | 往来寒热、胸胁苦满、默默不欲饮食、心烦喜呕、口苦咽干目眩 | 苔薄黄，脉弦 |
| **大柴胡汤** | 小柴胡汤证+腹满痛、便秘 | 苔黄，脉弦数 |

### 温里剂

| 方剂 | 方证要点 | 舌脉 |
|------|---------|------|
| **理中汤** | 腹满吐利、口不渴、喜温按 | 舌淡苔白腻，脉沉弱 |
| **四逆汤** | 但欲寐、肢冷、下利清谷、脉微细 | 舌淡苔白滑，脉微细 |
| **真武汤** | 少阴水气：腹痛、小便不利、四肢沉重疼痛 | 舌淡苔白滑，脉沉 |

## 六、常见脉象速查

| 脉象 | 特征 | 主病 | 常见证型 |
|------|------|------|---------|
| **浮** | 轻取即得 | 表证 | 太阳 |
| **浮紧** | 浮而紧绷 | 表寒实证 | 麻黄汤证 |
| **浮缓** | 浮而松弛 | 表寒虚证 | 桂枝汤证 |
| **沉** | 重按始得 | 里证 | 太阴、少阴 |
| **沉弱** | 沉而无力 | 里虚寒 | 太阴脾虚 |
| **沉迟** | 沉而缓慢 | 里寒 | 少阴 |
| **沉实** | 沉而有力 | 里实热 | 阳明腑实 |
| **数** | 一息六至以上 | 热证 | 阳明 |
| **洪** | 来盛去衰 | 热盛 | 阳明经热 |
| **滑** | 往来流利 | 痰/食/热 | 阳明、痰热 |
| **弦** | 如按琴弦 | 肝郁/痛 | 少阳、厥阴 |
| **细** | 脉细如线 | 血虚/阴虚 | 少阴阴虚 |
| **微** | 极细极软 | 亡阳 | 少阴亡阳 |

## 七、常见舌象速查

| 舌象 | 主病 | 常见证型 |
|------|------|---------|
| **舌淡苔白** | 寒/虚 | 太阴、少阴 |
| **舌淡苔白腻** | 寒湿 | 太阴脾虚湿盛 |
| **舌淡胖齿痕** | 阳虚水湿 | 太阴/少阴阳虚 |
| **舌红苔黄** | 热 | 阳明 |
| **舌红苔黄燥** | 热盛伤津 | 阳明经热 |
| **舌红苔黄腻** | 湿热 | 阳明湿热 |
| **舌绛无苔** | 阴虚/热入营血 | 少阴阴虚 |
| **舌紫暗瘀点** | 瘀血 | 厥阴瘀血 |

## 八、问诊十问速查

### 一问寒热

| 表现 | 辨证 |
|------|------|
| 恶寒发热同时 | 太阳表证 |
| 但热不寒 | 阳明里热 |
| 寒热往来 | 少阳 |
| 但寒不热 | 少阴/太阴里寒 |

### 二问汗

| 表现 | 辨证 |
|------|------|
| 无汗恶寒 | 太阳表实（麻黄汤） |
| 有汗恶风 | 太阳表虚（桂枝汤） |
| 但头汗出 | 少阳 |
| 大汗大渴 | 阳明 |
| 自汗（白天） | 气虚不固 |
| 盗汗（夜间） | 阴虚内热 |
| 冷汗淋漓 | 少阴亡阳 ⚠️ |

### 三问头身

| 表现 | 辨证 |
|------|------|
| 头痛（后脑） | 太阳 |
| 头痛（前额） | 阳明 |
| 头痛（两侧） | 少阳 |
| 头痛（巅顶） | 厥阴 |
| 身痛 | 太阳表证 |

### 四问便

| 表现 | 辨证 |
|------|------|
| 便秘干硬 | 阳明腑实 |
| 便秘不干费劲 | 气虚/阳虚推动无力 |
| 大便稀 | 太阴脾虚 |
| 先干后稀 | 脾虚兼积滞 |
| 水样便 | 太阴寒湿/少阴下利 |

### 五问饮食

| 表现 | 辨证 |
|------|------|
| 能吃但胀 | 太阴脾虚不运 |
| 不想吃 | 太阴脾胃虚弱 |
| 饿得快吃了难受 | 胃强脾弱 |
| 口苦 | 少阳 |
| 口淡 | 太阴 |
| 口渴喜凉 | 阳明热 |

### 六问胸腹

| 表现 | 辨证 |
|------|------|
| 胸胁苦满 | 少阳 |
| 腹满痛拒按 | 阳明腑实 |
| 腹满喜温按 | 太阴脾虚 |
| 少腹冷痛 | 厥阴/少阴 |

### 七问渴

| 表现 | 辨证 |
|------|------|
| 口渴喜凉 | 阳明热 |
| 口渴不欲饮 | 水饮/阳气不化 |
| 口不渴 | 寒证 |
| 渴喜热饮 | 里寒 |

### 八问旧病

> 问既往病史、用药史，判断体质、排除禁忌

### 九问因

> 问发病诱因、加重缓解因素

### 十问妇儿

**女性经带：**
| 表现 | 辨证 |
|------|------|
| 月经后期色淡量少 | 太阴血虚 |
| 月经前期色红量多 | 阳明血热 |
| 痛经有块 | 厥阴瘀血 |

## 九、合病速查

| 合病 | 表现 | 选方 |
|------|------|------|
| 太阳阳明合病 | 发热恶寒+口渴汗出 | 葛根汤、麻杏石甘汤 |
| 太阳少阳合病 | 发热恶寒+口苦胸胁苦满 | 柴胡桂枝汤 |
| 少阳阳明合病 | 口苦胸胁苦满+便秘 | 大柴胡汤 |

## 十、急危重症速查 ⚠️

> 辨证前先排除危险情况，中医不替代西医诊断。

| 表现 | 可能情况 | 处理建议 |
|------|---------|---------|
| **持续剧烈腹痛拒按** | 急腹症（阑尾炎、穿孔、肠梗阻） | 立即就医，排除外科 |
| **高热不退伴神志改变** | 重症感染、脑膜炎 | 立即就医 |
| **突发胸痛伴大汗气短** | 心梗 | 立即就医，拨打急救 |
| **突发剧烈头痛呕吐** | 脑出血、脑卒中 | 立即就医 |
| **呼吸困难喘促不止** | 重症肺炎、哮喘、肺栓塞 | 立即就医 |
| **呕血、黑便** | 上消化道出血 | 立即就医 |
| **但欲寐、肢冷、脉微欲绝、冷汗** | 少阴亡阳 | 急用四逆汤抢救+送医 |
| **意识模糊、抽搐** | 颅内病变、电解质紊乱 | 立即就医 |

## 十一、用方禁忌速查 ⚠️

| 方剂 | 禁忌 | 原因 | 替代 |
|------|------|------|------|
| **麻黄汤** | 高血压、心脏病、失眠 | 麻黄升压、兴奋心脏 | 荆防败毒散 |
| **麻黄汤** | 体虚、老人、孕妇 | 发汗伤正 | 桂枝汤/参苏饮 |
| **承气汤** | 孕妇、体虚、老人 | 攻下伤胎/伤正 | 润下/增液承气 |
| **四逆汤** | 真热假寒、阴虚 | 热证用热药加重 | 养阴清热方 |
| **白虎汤** | 表证未解 | 清里引邪入里 | 先解表 |

### 特殊人群

| 人群 | 注意 |
|------|------|
| 孕妇 | 禁攻下、破血；慎活血、滑利 |
| 老人 | 剂量宜轻，慎攻伐 |
| 小儿 | 按体重计，药性平和 |
| 体虚 | 攻伐减量+扶正，中病即止 |

## 使用建议

1. **打印随身携带** - 临床查阅
2. **对照病案练习** - 每看一个患者，用速查表核对
3. **逐步积累** - 遇到新情况，补充到个人速查表
